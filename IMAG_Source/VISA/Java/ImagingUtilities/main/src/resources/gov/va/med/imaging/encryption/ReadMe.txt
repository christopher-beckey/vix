1. To run the program
	a) open a command window
	b) in the same command window, change to the folder where the ImageUtility.jar resides
	b) enter the command: java -jar ImageUtility.jar -operation=[upgrade|downgrade|encrypt|decrypt]
	    -input=[full path to control XML file|text to encrypt/decrypt] -password=[password]

1. Pre-requisite for Upgrade/Downgrade operations:
	a) An already-made up "control" XML file that follows the given sample XML file must exist and full path is known prior to running this class.
	b) Java SDK must be installed and configured to run from anywhere
