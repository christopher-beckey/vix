faviconForEditing.png - Edit this image in your image editor of choice, such as MS Paint

favicon.ico - Open faviconForEditing.png in your icon creator of choice, such as irfanview, and save as this file
              Irfanview:
                Options > Properties/Settings > Viewing > Main window color, choose the white background color
                Image > Resize to 70 pixels
                File > Save As, Check Save Transparent color under PNG and ICO, Uncheck Save Transparency as Alpha channel, Check Use main window for transparency
