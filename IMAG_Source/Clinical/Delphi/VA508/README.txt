HOW TO INSTALL VA 508 Delphi Component

1/ Make sure the source codes are in correct folder C:\DevD2007\VA508

2/ Installing VAShared2006 Component
- Open VAShared2006.dproj
- Check for directory in project options. Make changes if you use a different folder location.
- Compile and Build the project
- Install the component

3/ Installing VA508Accessibility
- Open VA508Access2006.dproj
- Check for directory in project options. Make changes if you use a different folder location.
- Compile and Build the project
- Install the component
