Usability Enhancements

Customize display of configuration buttons

User-defined configuration buttons displayed near the top of the Capture window can now be re-ordered by dragging them with the mouse. When this method is used, the new button order is saved when the application closes.

 

Buttons can also be reordered using the Configuration List window (Configuration | Configuration List) by clicking column headings to sort list entries or by dragging list entries with the mouse. If this method is used, the user will be prompted to save the order when the Configuration List window is closed.

 

New icon legend help

The Capture application now uses more icons as a visual indicators of the current process or capture function. The meaning of most icons can also be displayed by pointing to them with the mouse.

 

A new Icon Legend dialog has also been added. This dialog displays the meaning of the new icons used in the Capture window.  This dialog can be opened by choosing Options | Icon Legend.

 

 

Show/hide Image Long Description area

The Image Long Description area displayed near the bottom of the Capture window can now be shown or hidden by clicking a new expand/collapse icon. To conserve display space, the Image Long Description area is hidden by default.

New Image menus

A new Image menu has been added to the main Capture window, and a new shortcut (right-click) menu has been added to the image display area in the Capture window. These menus provide an alternate way to access image manipulation functions that previously could only be accessed using the Image toolbar.
