Batch Capture/Import Enhancements

Import directory included in saved configuration

When a Capture Configuration for imports is saved, the current import directory is saved with the configuration. When the saved configuration is selected by a user, the import directory is automatically selected in the main Capture window.

 

Define and manage a list of import directories

A new menu option, Tools | Import Options | Import Directory Options, has been added. Using this option will open a dialog where the user can create and manage a list of workstation-specific import directories.

 

 

Quickly change selected import directory

Once a saved list of import directories is defined, users can quickly switch from one directory to another using a new expand/collapse icon in the shown near the active import directory in the Capture window.

 

 

To change the active import directory to a different saved directory, a user clicks the expand/collapse icon. This replaces the existing list of files with a list of saved import directories. The user can click a directory in the list, and then click the expand/collapse icon again to make the selected directory the active directory.

 

 

Status/management Icons for batch capture/import

New icons in the import/batch capture list indicate the state of each image in the list as the images are imported. New icons at the top of the list can be used to manage the contents of the list.
