Other Progress Notes Window Changes

New filtering options for select progress note list

In the button bar used to filter the list of available Progress Notes, the Signed, Unsigned, and Uncosigned buttons can be used in any combination (previously, only one of these buttons could be active at once).

 

 

A user can also use the new My Notes button to list only their own notes.

 

New preferences for progress notes

The user can now customize aspects of the Progress Notes dialog such as the number and date range of notes to retrieve, and whether or not to use status icons in the notes listing and in the main Capture window. The Progress Note List Options dialog can be accessed from the Progress Notes dialog by clicking Options | List Options, or by choosing the Select Note tab and clicking the new List Options button.

 

 

New note listing status bar

A new status bar has been added above the listing of available progress notes. The status bar displays the total number of notes listed, the counts of each selected status, the total number of notes listed, and the selected date range.

 

 

 

Consults

If the Title of the new Note is a 'Consult' Title VistA Imaging will display a list of Patient Consults and require that the user make a selection.

 

When the New Note is successfully saved, and the Status of the New Note is Signed or Electronically Filed, the Consult status will be updated to 'Complete'.

 

 

Show Related Notes / Addendum

 

 

When the menu option ‘View|Show Related Notes/Addendum’ is checked,

  • all Notes and Addendums related to the listed Notes

  • and Addendums are displayed in a ‘Tree Style’ list.

 

(Addendums are listed directly under the Note and are indented)

 

The graphic below shows a list of completed notes/addendums for a patient, when the menu option:

  • 'Show Related Notes/Addendums' is Not checked.

 

(A list of completed Notes and Addendums for the patient, sorted by date.)

 

The next graphic shows a list of completed notes/addendums for a patient when the menu option 'Show Related Notes/Addendums' Is checked.

 

(All Notes/Addendums,  Signed and Unsigned, are now displayed.)

 

When this view is selected (i.e. Show Related Notes/Addendums is Checked.)

  • Sorting of the columns is disabled.

  • Either Signed or UnSigned can be selected, but not both.

 

If UnSigned Notes are listed, sorting is by the greater of:

  • The date of the Unsigned Note

  • Or the date of any of it’s unsigned Addendums

 

If Signed Notes are listed, sorting is by the greater of:  

  • the date of the signed Note

(the date of any Addendum is not considered)

 

 

Workload Credit

In Patch 59, users of VistA Imaging Capture can create new Notes to associate with the Scanned Documents. By default, the visit status of new notes created in VistA Imaging Capture is set to 'Historical'. This ensures that the visit is not used for workload credit and documents captured via Patch 59 will not incur add/edit errors in PCE encounters.

 

 

Historical visit

Historical visit status is used for documenting a visit that occurred some time in the past (exact time may be unknown) or at some other locations (possibly non-VA) and is not used for workload credit, they can be used for setting up the reminder maintenance system.

  • By default, visit status via scanning in Patch 59 is set to historical. This ensures that the visit is not used for workload credit for documents captured.

 

Alerts

Alerts are sent when:

  • The Image Capture process is successful and:

(a) the person scanning the image is the ordering provider, or

(b) an Orderable Item (OI) Alert has been set up by the CAC for the person scanning the image, or

(c) the person scanning the image is on the Consult Team.

 

Tele-Retinal Alerts

If the Imager is Remote,

    • they will get an alert:

      • If the Imager is the Ordering Provider,

      • If not the Ordering Provider then the CAC needs to set up an Orderable Item Alert.

If the Imager is Local (in the same database),

    • they will get an alert:

      • If the Imager is the ordering provider, or

      • If an Oderable Item (OI) Alert has been set up by the CAC, or

      • If the Imager is on the Consult Team.

 

Capture Client Alerts

Alerts are sent according to the same criteria as CPRS.

 

Consults

When a 'Consult' Note is created in VistA Imaging Capture the user must select from a list of Patient Consults.

The Consult is marked as 'Complete' when:

(a) The Image Capture process is successful, and

(b) The resulting status of the Note is 'Complete' by signing, or electronically filing the Note.
