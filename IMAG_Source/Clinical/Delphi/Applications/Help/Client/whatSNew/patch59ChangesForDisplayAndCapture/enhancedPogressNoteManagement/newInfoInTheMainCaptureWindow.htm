New Info in the Main Capture Window

The following new features and changes made to the Clinical Capture software make it easier and faster to work with Progress Notes.

 

Before Patch 59, the Capture window would show the following information about a selected Progress Note:

 

 

After installing Patch 59, this area will show increased detail about the selected Progress Note:

 

 

(1) The title of the selected note is now displayed, instead of the generic 'Study Performed: Note' label.

 

(2) A new ‘Note-Status-Loc’ field uses icons* and/or text to indicate the present state of the selected note, the intended status of the note and the location of the note.

 

(3) A Visit Location setup button. Clicking the button will open the 'Visit Location' tab in the Options Window.

 

Users can:

(a) Select to not use defaults and select a visit location each time a new note is created.

 

(b) Select to use defaults.

  • The Visit Location from the current form will be used if it is defined.

or

  • The User's own default Visit Location will be used.

or

  • The default Visit Location defined for the workstation will be used. A user must have the MAG SYSTEM Key to define a default Visit Location for the workstation.

 

(4) The new icons* to the left of the Select Progress Note button indicate if any text or boilerplate text exists for the note.

 

* Selecting Options | Icon Legend | Note Icons will display a dialog that lists what each icon means. The user can also choose if icons or text should be used to indicate a selected note’s status.
