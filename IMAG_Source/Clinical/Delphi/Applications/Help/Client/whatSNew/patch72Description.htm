Patch 72 Description

PDF Viewing

In Patch 72, PDF documents are now viewed using the Full Resolution viewer instead of using the Adobe Reader.

  • This allows users to compare PDF documents side by side with each other and other images in the patient records.

Note: A feature to be added will allow the user to decide if they want to use the Full Resolution viewer to display PDF documents or if they want to use the Adobe Reader to display PDF documents. This option will be stored as a user preference.

 

DOD Images

The patch 72 Display client can view DOD images by communicating with a local VistA Imaging Exchange (ViX) server.

  • If a site does not have a ViX server they will be unable to view DOD images.

    • For patients who have been seen at the DoD, a DoD button will appear on the main Display client window in the Remote Image View toolbar.

    • a DOD button will appear on the main Display client window in the Remote Image View toolbar.

 

When the user selects to connect to the DOD:

  • the Display client communicates with the local ViX server to retrieve DOD studies and images.

  • DOD studies are displayed for the user in the same format as VA images (in the image list window and the abstract window).  

Studies are sorted by procedure date with the most recent studies appearing first.

 

Users can use the same tools to view DOD images as they would use to view VA images including all of the functionality of the Radiology Viewer.

Radiology Viewer - Magnify Pan Window

Use the pan window to zoom in on a small portion of the image.

Click on the Pan Window button on the toolbar .

The button will be depressed, and the small Pan Window will appear next to the radiology window.

Click and drag the mouse down to move the selected area of the image to be shown.  The mouse button can be released before moving the mouse.  

Then when the mouse is clicked, the magnify window will show the image region near the cursor.

  • Clicking on the button will close the magnify window.
  • The magnify window will also close when the Pan Window dialog is closed.

If you click on another image, it will be selected.

Clicking again will activate the Magnify function for the new selected window.

Measuring

Measurements can now be made directly on the image being viewed.

  • To take measurements press the button on the toolbar to activate the ruler tool.

Note: For certain images the button might not be enabled. This is because the informtion necessary to accurately calculate the measurement is not available in the image information.

 

Use the mouse to place the cursor over one end of the object to be measured.

    • Click down with the left mouse button.

    • Hold the left mouse button down while moving the cursor to the other end of the object to be measured.

    • Release the mouse button.

    • A line will appear to indicate the length measurement that was made.

    • The measurement will appear on one side of the ruler line.

 

Measuring Angles
If angle is selected, the user will trace the angle.

  • Use the mouse to place the cursor at the point of the vertex of the angle to be measured.

  • Click down and up with the left mouse button. 

  • Move the cursors to end of one of the line segments and click down and up with the left mouse button.

  • Place the cursor over the end of the second line segment of the angle to be measured,

  • and click the left mouse button again.

A line will appear to indicate the angle measurement that was made.

  • The measurement will appear near the vertex of the angle.

Multiple measurements may be made; the results of each will be displayed.

    • To clear the lines from the image, select the 'Tools' menu, then select 'Ruler/Angle Tool' and select 'Clear All Measurements'.

    • The user can also use the button to select measurments and move them or delete them by pressing the Delete key or from the 'Tools' menu, select 'Ruler/Angle Tool' and select 'Delete Selected Measurements'

      • Note: Measurements are not kept when changing between images. Once a new image is displayed, the measurments from the image are removed.

    Measurement Options

From the ‘Tools’ menu, select ‘Ruler/Angle Tool’ and select ‘Measurement Options’ to display the    Measurement Options dialog. 

       This allows the user to change the line width, the measurement units and the measurement color. 

  • Pressing the ‘Apply’ button
    • will apply the changes and keep the dialog open to allow further modifications. 
  • Pressing the ‘OK’ button
    • will apply the changes and close the dialog. 
  • Pressing the ‘Cancel’ button
    • will cancel any changes made. 

These settings are saved as user preferences for future Imaging sessions.

Menu

The menu options in the new Radiology Viewer are almost identical to the old Radiology Viewer.  This was done to make it easier for people to adjust to the new viewer.  

 

 

Copy: This allows the user to copy the image to the clipboard.  The user must provide their electronic signature and give a reason why they want to copy the image.

Print: This allows the user to print the image.  The user must provide their electronic signature and give a reason why they want to print the image.

Report: This displays the report associated with the image.

Exit: This closes the Radiology Viewer. 

 

Invert (Reverse): This inverts the colors of the image.

Reset Image: This resets the currently selected image to its original display settings including zoom, window/level and orientation.

Reset All: This resets all of the images to their original display settings including zoom, window/level and orientation.

Cache Selected Study: This caches the currently selected study in the background to the workstation.  This allows the user to continue working while the images are being brought to the workstation.

 

Rotate Clockwise 90: Rotates the currently selected image clockwise (to the right) 90 degrees.  If “Apply-All” is selected, then all images will rotate.

Rotate Minus 90: Rotates the currently selected image counter clockwise (to the left) 90 degrees. If “Apply-All” is selected, then all images will rotate.

Rotate 180: Rotates the currently selected image 180 degrees. If “Apply-All” is selected, then all images will rotate.

Flip Horizontal: Flips the currently selected image horizontally. If “Apply-All” is selected, then all images will flip horizontally.

Flip Vertical: Flips the currently selected image vertically. If “Apply-All” is selected, then all images will flip vertically.

 

 

Note: This menu is only available when viewing a CT study.

Abdomen: Sets the currently selected window/level values such that they can view the abdomen. If “Apply-All” is selected, then all images will change to the same window/level value.

Bone: Sets the currently selected window/level values such that they can view bone. If “Apply-All” is selected, then all images will change to the same window/level value.

Disk: Sets the currently selected window/level values such that they can view disk. If “Apply-All” is selected, then all images will change to the same window/level value.

Head: Sets the currently selected window/level values such that they can view the head. If “Apply-All” is selected, then all images will change to the same window/level value.

Lung: Sets the currently selected window/level values such that they can view the lungs. If “Apply-All” is selected, then all images will change to the same window/level value.

Mediastinum: Sets the currently selected window/level values such that they can view the mediastinum. If “Apply-All” is selected, then all images will change to the same window/level value.

 

 

Image Info: Displays information about the image, including the patient name, ID, number of pages in the image, the page dimensions, bit depth, and file format information.

Txt File Info: Displays the TXT file associated with the image (if it is available).

Radiology Image Info: Displays information from the header of the image including the modality, series and image numbers.

 

 

1x1 Stack: Puts the viewer into a single image stack mode.

2x1 Stack: Puts the viewer into a dual stack mode where two images from different (or the same) studies can be viewed and compared.

3x2 Layout: Puts the viewer into a layout mode where 6 images are displayed at once.

  4x3 Layout: Puts the viewer into a layout mode where 12 images are displayed at once.

 

 

Stack View: Contains options for viewing with the stack view mode.

Layout View: Contains options for viewing with the layout view mode.

Image Settings: Contains settings for how images are displayed when first loaded.

Labels On: If checked, then the orientation labels are displayed on the image (always on the middle left side of the image).

 

 

Page Together: When checked, two stack views will page together.

This does not guarantee they will always be on the same image number, but as one stack changes images, the other stack will follow.

Page With Different Settings: When selected, each stack will page with the settings held from the previous image in that stack.

Page With Same Settings: When selected, each stack will page with the settings from the stack that changed images.  

If the left stack changes images, both stacks will change images and then set their view settings (window/level, zoom, etc.) to the same as the left stack.

Page With Image Settings: When selected, each image will automatically set itself based on the contents of the image.  It does not matter what the previous image had set.

 

Use Selected Window Settings: When selected and the user changes images, the next image loaded will use the same settings as the previous image.

Use Individual Image Settings: When selected, each image will automatically set itself based on the contents of the image.  

It does not matter what the previous image had.

 

 

Use Device Window Level if Available: When selected, the image will be automatically window leveled using values specified in the DICOM header (if available).

Use Histogram for Window Level: When selected the image will be window leveled using the contents of the image.

 

 

Cine Tool: Displays the cine tool dialog to allow moving between frames in a single image.

Ruler/Angle Tool: Contains options for using the ruler and angle tool.

Pixel Values: When checked, if the user clicks on an image, the hint will change to the value of the pixel at the location clicked.  

If not checked, then the hint will contain series and study description information.

 

Enable Ruler: Turns on the ruler tool (only enabled if the image contains the necessary information to make accurate measurements).

Enable Angle Tool: Turns on the angle measurement tool.

Ruler/Protractor Pointer: Changes the current tool to the pointer for measurements.  This allows the user to move and modify a measurement mark.

Delete Selected Measurements: Deletes any measurements the user has selected.

Clear All Measurements: Clears all measurements from the image.

Series Drop Down List

The series drop down list shows the series included in the current study.  

  • The user can select a series in the list to move to the first image in that series.  

  • If the description is too long to fit into the box, the hint of the drop down list will show the entire description of the selected series.

Note: The description of a series is only shown when an image in the series has been selected (because the series description information comes from the image header).  As a result, the series drop down list might not contain the series description when the study is first loaded.  

    • The series drop down list
      • will automatically populate itself when the required information is available.
      • will not appear if series information is available in the study.

The side toolbar has two parts.

Below the buttons is a study slider that shows the position of the image in the current study.

  • The button below the slider allows the user to play the images in the study (in a loop).

  • When the user presses the play button it will change to a stop button

    • to allow the user to stop the scrolling of images.

  • Below the button is another slider that controls the speed of the loop.

 

 

 

 

The user can right click on the study slider and view a menu with options:

 

  • Range Start: Set the range of the loop to start at the currently selected image.

  • Range End: Set the range of the loop to end at the currently selected image.

  • Range Clear: Clear the range of images to scroll.

  • Start: Start the looping (same as pressing the play button).

  • Stop: Stop the looping (same as pressing the stop button).
