Remote Image Views User Preferences

Also: See Topic 'View Preferences window' for a description of all options available to local viewers of VistA Imaging Display.

 

The user preferences dialog has been reorganized for better use of screen space. The 'When a patient is selected’ and ‘Abstract and Group Windows’ tabs display existing preferences. The ‘Remote Image View’ tab is new to this patch and is explained below.

 

To access the user preferences click on the ‘Options’ menu and click ‘View Preferences’

  • Patch 45 introduces many new user preference options to the user.

  • These user preferences are saved to the VistA database.

  • To save a user preference for the current session the user should make the changes and click Close.  This will automatically close the dialog.

  • To save a user preference to the VistA database, the user should make the changes and click Save.

 

Remote Image Views Auto-Connect Enabled

  • Selecting this option makes the Display client automatically connect to remote sites based on the selected patient.  For each site the patient has visited, the client will attempt to connect to that site and retrieve images.

  • If this option is disabled, sites will not automatically be connected.  However, they will appear in the Remote Image Views Configuration dialog site list.

  • This option is enabled by default.

 

Only Auto-Connect to Sites in Local VISN

  • Selecting this option makes the Display client automatically connect only to remote sites in the user’s local VISN. The sites to be connected are based on the sites the patient has visited. If the patient has visited a site not in the local VISN, the site is not automatically connected.

  • If this option is disabled, then the sites inside and outside of the VISN will be connected based on the patient selected.

  • ‘Remote Image Views Auto-Connect Enabled’ must be enabled for this option to be available.

  • This option is disabled by default.

 

Hide ‘Patient Active’ Sites With 0 Images on Toolbar

  • Selecting this option causes only sites where the patient has at least one image study to appear on the Remote Image Views toolbar on the main imaging window.

  • If this option is disabled, then all sites where the patient has visited will appear on the Remote Image Views toolbar on the main Imaging window regardless of how many image studies were performed on the patient at that specific site.

  • This option is disabled by default.

 

Hide Disconnected Sites For Selected Patient on Toolbar:

  • Selecting this option causes only currently connected sites where the patient has visited to appear on the Remote Image Views toolbar on the main Imaging window.  If there was an error connecting or if a site was disconnected, then that site will not appear on the Remote Image Views toolbar.

  • If this option is disabled, then all sites where the patient has visited will appear in the Remote Image Views toolbar on the main imaging window regardless of the status of the remote site connection.

  • This option is disabled by default.
