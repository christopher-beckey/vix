Remote Image Views Configuration Dialog

Opening the Remote Image Views Configuration Dialog

 

(1) Click the Remote Image Views Configuration button on the main Imaging window. (Log in to the VistA server to enable this option.)

(2) The Remote Image Views Configuration dialog will appear.

 

 

(3) To close the dialog click the Close button.

  • Sites for selected patients will appear in the dialog along with the status of each site. Sites can have a status of Patient Active, Patient Inactive, or Disconnected.

  • The number of image studies from the site for the selected patient is also displayed in the list.

 

Disconnecting a Remote Site

(1) Select a site with status ‘Patient Active’ or ‘Patient Inactive’ in the site list and click disconnect.

(2) This will cause the site to be disconnected and the display client to reload the images for the patient.

 

Connecting a Remote Site

(1) Select a site with status ‘Disconnected’ in the site list and click Connect.

(2) This will cause the site to be connected and the display client to reload the image information for the patient.

 

Disconnect All Remote Sites

(1) Click Disconnect All to disconnect all remote sites.

(2) This will cause all remote sites to be disconnected and the display client will reload the images for the patient.
