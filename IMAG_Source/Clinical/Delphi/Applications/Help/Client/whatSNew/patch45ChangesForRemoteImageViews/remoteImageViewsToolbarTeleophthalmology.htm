Remote Image Views Toolbar

For patch 45 there is a new toolbar on the main Imaging window. This toolbar informs the user of the remote sites the patient has visited and how many Image studies the patient has at that remote site.

As shown above, the patient has been seen at two remote sites: Washington and Togus. The user is currently connected to Washington but is not connected to Togus.  The patient has 4 images/studies at Washington.

 

Connecting All Disconnected Sites for Current Patient

If any remote sites for the selected patient are not connected you can connect all disconnected remote sites for the current patient.  If all remote sites for the current patient are connected or if the patient has no remote sites, the Connect All button will not appear.

 

(1) Left Click on the Connect All button to connect all disconnected remote sites for the current patient.

(2) The patient’s images will refresh when all of the remote sites have finished connecting.

 

Connecting Individual Disconnected Sites

If a site appears in red in the Remote Image Views toolbar you can manually connect to the site.

 

(1) Left Click on the site you wish to connect to.

(2) This will cause the site to be connected and the display client to reload the image information for the patient.
