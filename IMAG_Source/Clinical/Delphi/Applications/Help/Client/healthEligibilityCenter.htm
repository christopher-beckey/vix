Health Eligibility Center (HEC)

Sending Means Tests to the HEC

 

The Health Eligibility Center has requested that a 3rd type of Means Test be copied to them.

 

Sites need to edit the Image Actions file and add the third image Index Type.

MEANS TEST (10-10EZR)

 

The following is the current list of 'Image Types' that need to be sent to the HEC.

MEANS TEST (10-10EZ)

MEANS TEST (10-10EZR)

MEANS TEST (10-10F)

 

A qualified person at the site needs to use VA Fileman to edit the IMAGE ACTIONS FILE: and choose HEC COPY at the Image Action name field prompt.

 

Note: Alternatively, you can log a Remedy Ticket and have VistA Support guide you. Also, to resend documents that have failed, or documents that were never sent to the HEC, you'll also need to enter a Remedy Ticket.

 

See VistA Imaging System Technical Manual for an example of "Adding a New Index Type to be sent to HEC".
