Sign-on window

Enter the appropriate Access code into the Access Code input field.  Press <TAB> and enter the appropriate  Verify Code into the Verify Code input field. Click on the 'OK' button or press <TAB> to move to the 'OK' button and press <Enter>.

 

Verify codes are originally assigned by the Information Resources Management (IRM) Service at the site; but the verify code must be changed periodically.

 

Press the Cancel button if it is not possible or it is not desired to sign onto VistA.
