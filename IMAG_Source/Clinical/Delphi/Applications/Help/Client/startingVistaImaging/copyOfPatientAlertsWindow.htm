Patient Alerts window

The Patient Alerts window displays the Restricted Patient Record , Similar Patient Record and Means Test Required alerts.

 

Restricted Patient Record

VistA Imaging displays the Restricted Patient Record Window if a patient is selected with a record that is restricted to users who can demonstrate a need to know the information in the record.

 

If the 'OK' button on this window is selected, the station Security Officer will contact the user and ask the user for justification for accessing this record. Requests for restricted patient records are tracked by VistA Imaging.

 

Similar Patient Records

This alert will display if the Last name and Last 4 numbers of the SSN are identical to another patient in the VistA System.

 

 

Means Test Required

Site specific information describing what to do when a Means Test is required will be displayed.
