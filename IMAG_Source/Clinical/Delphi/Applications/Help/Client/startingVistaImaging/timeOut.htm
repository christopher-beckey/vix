Time Out

The VistA Imaging Display Window, and VistA Imaging Capture Window will timeout and close if they have been inactive for a certain length of time.

 

The timeout values are maintained by the System Manager and can be different for each application. The System Manager can also override the site timeout and set a workstation specific timeout value.

 

When the timeout has been reached the user will have 30 seconds to 'Cancel' the timeout and stay connected to VistA. Clicking the 'OK' button will force the timeout and close the application
