VistA Imaging Capture is the main window for adding images and documents to the patient record.

 

Other windows will be opened during the capture process to:

  • select, confirm and edit the information being stored.

 

The program provides many options for customizing the Capture window.

  • when these settings are saved, the user's window preferences will be retained on all workstations at that site.

 

1 - Begin by selecting a patient.

  • If a Photo ID of the patient was stored it will be automatically displayed.

  • Patient ID data will be shown to the right of the patient's name.

 

2 - Click on a configuration button - in the case (see the window above) it was the Medicine button.

 

3 - Click the Select Medicine Procedure button.

The images captured will be Associated with the VistA Medicine package and attached to a procedure.

 

4 - From the Select Medicine Procedure window choose a medical procedure (either a new one or a previous procedure) to attach the images.

  • The selection window will close.

 

5 - In the VistA Imaging Capture main window, notice that the fields Study Performed and Procedure Date have been filled.

  • The example above has "COL" (colonoscopy) in the Study Performed field.

 

6 - Click the down-pointing arrow on each of the following fields -- in order, from top to bottom:

  • Doc/Image Type

  • Specialty *

  • Proc/Event *

  • Image Desc(ription)

  • an asterisk following the name of the field means that it is 'optional'

 

The order of selection is important because the 'first' entries determine the contents of fields listed in the 'following' entries.

 

7 - Proceed with the capturing of images/documents to the patient record.

  • After each 'request' to capture an image, the user is required to review the accuracy of the image in the Display area and the correctness of the patient information.

  • If an image is not satisfactory, click the Cancel button.

    • If an image is satisfactory, click the Image OK button -- to save the image in the patient record.

 

8 - At completion of the Study (Group) capture session, click the Study Complete button.

  • A Long Description can then be entered before finishing the Study.

 

9 - Display images/documents saved in the patient record.

  • Click the File menu (top left of Capture window).

  • Click the option - Latest Patient Images.

  • From the Patient's Latest Images list double-click the selection.

    • View the image in the Display area.

 

Notes:

  • 'Clear Input Field' Behavior

    • When a single image is successfully captured, all fields will be cleared.

    • When the batch capture of a group of images is completed, all fields will be cleared.

    • When an individual image within a group is captured, all field values are retained.

 

Users can alter this behavior by using the Hold Value feature.
