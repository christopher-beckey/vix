Notes:

  • Entering free text into the edit box of the drop down list is possible but not recommended. If there isn't an appropriate category in the drop down list, it is preferred that the System Manager be contacted.

  • VistA Imaging has the optional functionality of limiting a user's ability to capture images to certain packages based on assigned Imaging Capture Keys.

  • If a dialog window appears with a message similar to: 'You don't have the proper Security Keys to capture Medicine Images', Then contact the system manager.

  • If images are being captured for a Study Group and the Study is not yet complete, all Association choices will be disabled. The previous Study must be closed by clicking on 'Study Complete' before a new Association can be selected.
