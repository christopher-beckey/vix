Abstracts and Group Abstracts-buttons

See also: Abstracts and Group Abstracts, Abstracts and Group Abstracts-shortcut menu

 

When the Abstracts or Group Abstracts windows are opened, all available images (or their proxies) are loaded. Scroll bars and paging buttons can be used to view any additional abstracts.

  • The title bar in each window reflects the total number of images.

  • The toolbar shortcut menu, opened by right-clicking the toolbar, can be used to show, hide, or position the toolbar.

 

The Previous Page button shows the previous page of abstracts on the Abstracts window.

  • This button will be disabled when the loaded abstracts include the first abstract.

The Next Page button shows the next page of abstracts on the Abstracts window.

  • This button will be disabled when the loaded abstracts include the last abstract.

The Decrease Abstract Size button decreases the size of the abstracts in the Abstracts Window.

  • The size of the abstracts is saved as a View Preference.

The Increase Abstract Size button increases the size of the abstracts in the Abstracts Window.

  • The size of the abstracts is saved as a View Preference.

The Refresh button reloads the abstracts from the Imaging Network Servers.

  • Use this button when the abstracts are not displayed properly.

  • Also, when an Imaging Server that wasn't available or reachable has been fixed.
