Patch 72 Overview

Patch 72 upgrades the imaging component used within the Clinical Display client to AccuSoft ImageGear 15.4 (currently).

  • This component is now used throughout the application for viewing of all images.

  • By using this new component we have fixed several problems we had with the previous version of AccuSoft ImageGear (IG99) including:

    • the odd-columns problem and viewing embedded thumbnails.

 

Included in this patch is a brand new Radiology Viewer that makes use of some of the features available in this new image toolkit.

  • This new viewer is based on the functionality of the old Radiology Viewer but has a look and feel that matches more closely to the Full Resolution viewer.

  • The goal here is to make it easier for users to access buttons and functions.

 

  • Also in this patch are new components needed to communicate with the DOD and view DOD images.

    • These components allow images and studies from the DOD to appear like VA sites.

    • If a patient has been seen at the DOD, the DOD site will appear on the Remote Image View toolbar in the same manner as a VA site would.

    • Communication with the DOD is done through the VistA Imaging Exchange (VIX) service (Patch 83).

      • A site can only view DOD images if they have a VIX installed.
