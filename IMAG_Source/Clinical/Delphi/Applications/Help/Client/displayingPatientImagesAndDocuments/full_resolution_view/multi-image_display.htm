Multi-image display

Also see: toolbar

 

Single or multiple images and documents can be:

  • displayed, added, selected, manipulated and removed from this window.

  • manipulation of images and documents changes only the displayed object (not the image or document stored in the patient record).

 

From the Abstracts and Group Abstracts windows click on those images that you want to display simultaneously in the Full Resolution View window.

To select multiple images in this window:

  •  hold down the Ctrl key and click on an image in this window.

  • each image selected will be displayed with a blue border.

  • buttons clicked or sliders adjusted -- will effect all selected images.
