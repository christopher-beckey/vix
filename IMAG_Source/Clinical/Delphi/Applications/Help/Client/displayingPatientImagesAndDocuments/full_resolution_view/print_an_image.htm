Print an Image

The print button is located on all Image Viewers. i.e. Full Resolution View, Document, MUSE EKG and Radiology and window tool bars. Clicking on this button will print the image being displayed.  The patient name and image description will be printed at the top of the page.

 

 

The Clinician Agreement for downloaded images Window will open when an Image Print / Copy is attempted. The purpose of the Image Print / Copy and an Electronic Signature are required before copying the image will be enabled.

 

The Menu option 'File|Print' is located on the Radiology view window. Clicking on this option will print the image being displayed.
