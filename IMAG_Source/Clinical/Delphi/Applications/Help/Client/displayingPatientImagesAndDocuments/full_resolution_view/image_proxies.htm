Image Proxies

See also: Image proxies - tips

Image proxies:

o     are available in the Abstracts, Group Abstracts and Full Resolution View windows,

o     appear temporarily,

o     are useful where network and memory are limited,    

o     appear as a white rectangle.

 

 

To replace:

o     an individual proxy with an image - click the proxy,

o     all proxies with images - resize or refresh the window.

 

The number of proxies appearing is based on:

o     the size of the window and Image Viewer Settings dialog.
