Full Resolution View - buttons

 

Documents are displayed using the Full Resolution View window.

  • If the image is a Multi-Page image, the paging controls will be visible.

 

The Slider controls for Brightness, Contrast and Zoom will increase or decrease the corresponding property of the Displayed image.

 

Redisplays the image undoing all rotate,invert,contrast,zoom, etc.

Enlarges image to fill the display box.

Enlarges image to fit the height of the display box.

Enlarges image size to fit the width of the display box

Each color is replaced by its color complement.

  • For example, white becomes black, and red becomes blue.

Rotates the image 90 degrees counterclockwise.

Rotates the image 90 degrees clockwise.

Click Mouse Pan button.

  • The mouse can then move the image within the window.

Click Pan Window button and a window appears.

  • The whole image is displayed in the Pan Window, moving the mouse within the pan window will move the image around inside the control.

Use the Zoom In On Rectangle button to select a rectangle within the document.  

  • VistA Imaging will enlarge the rectangle to fit in the display box of the window.

Copies the image to the Windows clipboard.

Opens the Windows print dialog enabling selection of a printer and print formats.

Displays the VistA Report for the image.

Apply actions to all images.

Zoom image to actual size (1 pixel to 1 pixel).

Flip image vertically; flip image horizontally.

Click to turn 'magnifying glass' on or off. Can display a portion of the image at higher magnification.

Maximize the current image (enlarge image to fill entire display area).

Tile all loaded images.

Close all images.

Close selected images.
