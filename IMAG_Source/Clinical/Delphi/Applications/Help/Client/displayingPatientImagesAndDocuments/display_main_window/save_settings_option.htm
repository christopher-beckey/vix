Save Settings Option

Saved

If the size or position of any window is changed, the new arrangement can be saved as the user preference.

  • Select 'Save Settings' under the Options menu to save settings now.

  • Check 'Save Settings On Exit' to have settings automatically saved when VistA Imaging is closed.

 

The settings that are saved:

  • Size and position of all open windows;

  • Size of the abstracts in the Abstract and Group windows;

  • Font of the Reports window.

  • Settings in the View Preferences window.

 

Settings for each user are stored in VistA. This feature enables the user to have the same settings on any imaging workstation.

 

The following settings are saved when a user selects either of the "Save Settings" options from the Options menu in the main Imaging Display window.

  • Toolbar settings in the Abstract, Abstract Group, and Image List windows.

  • Image Viewer Settings values (number of rows, number of columns, max visible settings, lock image size settings) for the Abstracts, Group Abstracts, and Full Resolution View windows.

  • Show/hide and column width settings in the Image List window.
