View Preferences window

Also: See Topic: Remote Image Views User Preferences (Teleophthalmology)' for a description of all options available to remote users of VistA Imaging Display.

 

VistA Imaging allows each user to decide how certain features of the Imaging system will function.

  • Select View Preferences under the Options Menu of the VistA Imaging System Window to open the tabbed dialogue.

  • When the 'Save' button is clicked the settings from these windows, along with other window settings that are user definable, will be saved as the User's View Preferences.

  • Changes take effect immediately and remain in effect until VistA Imaging is closed.

  • Settings for each user are stored in VistA. This feature enables the user to have the same settings on any imaging workstation.

 

Patient Selected

Click to insert 'check mark' next to each window that should be opened when the patient's name is entered in the main window.

 

This is the recommended setting for most clinical users.

  • Show Radiology Exams window

Lists all radiology exams for the patient and includes the:

  • day/case number, exam date/time, procedure, and status of report.

When images are available online for a case, an (I) will appear after the status report. VistA Imaging users can select an exam and view the report or images associated with that exam.

  • Show MUSE EKG's

If the VistA Imaging application is connected to a site that has the Marquette MUSE EKG system installed, then it is possible to view patient EKG's in the Imaging System. Each site has to purchase a license from Marquette, in order to view EKG's from the Imaging System.

  • Show Progress Notes

When checked, the Progress Notes selection window will open when a patient is selected.  Otherwise, this window will not appear automatically for all patients.

 

Abstract & Group windows

When the VistA Imaging TeleReader is accessed, clicking (check mark) this option will display the remote abstracts.

 

Remote Image View

These preferences configure the remote connection's display for the VistA Imaging TeleReader.

 

The 'Display Juke Box Abstracts' option

  • has been removed in V 3.0.8.

  • When users had this option enabled, unacceptable performance delays were occurring.  Now, abstracts that are on the jukebox are always indicated by the icon.

  • The system automatically queues any jukebox abstract to be copied to the Network Server by the Background Processor.
