Image Viewer Settings

 

The Image Viewer Settings Dialog can be used to define the current and default layouts of the Abstracts, Group Abstracts, and Full Resolution windows.

This dialog can be opened from the Full Resolution viewer by clicking View > Viewer Settings.

o     This dialog can be opened from the Abstracts windows by clicking

 

Settings in the Image Viewer Settings dialog are specific to the window that the Image Viewer Settings dialog is opened from --

 

A user can define the number of rows and columns to display using the Row and Column boxes, or by clicking the Row x Column menu and selecting a preset "layout."

Note: the Column box is disabled for the Abstracts and Group Abstracts windows, which have a preset height/width ratio.

 

Determines the number of abstracts/images to display. Additional images will be displayed as proxies.

Note: The Abstract and Group Abstracts windows have an Auto Load function that increases the number of displayed Abstracts, if the number of displayed images (rows * columns) is greater than the value of Maximum Visible Count.

 

o     This essentially means that if the software has to choose between displaying an abstract or a proxy, it will choose to display the abstract.

 

Determines how images are adjusted when the window is resized.

o     When this option is checked, resizing the window will cause additional rows or columns to be shown. The current image size will be retained.

o     When this option is not checked, resizing the window will cause images to be resized to fit the available space. The number of displayed rows and columns will be retained.
