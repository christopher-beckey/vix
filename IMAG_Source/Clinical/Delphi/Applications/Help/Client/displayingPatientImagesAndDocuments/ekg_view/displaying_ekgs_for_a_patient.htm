Viewing Studies

 

 

From the VistA Imaging Display application, choose a patient and view the MUSE EKG window.

 

This will open the EKG Display window and load the list of studies available for the selected patient. On the right side of the screen, the list of studies should drop down.

 

 

The VistA Imaging Display application can also be configured to show the EKGs for a patient automatically whenever a patient is selected.  This is done through the View Preferences menu item located in the Options menu.

Grid

The grid provides an easy way to measure and compare portions of the study.  Studies can be displayed both with the grid on and off.  To toggle the grid, click the Display/Hide Study Ttext Overlay button .

BACK

Study Text Overlay

Studies can have an overlay of text that contains differentiating data that is always on the screen.  This can be used to prevent confusion over which study is which when viewing multiple studies concurrently.  To toggle this text on/off, click the Display/Hide study text overlay button.

BACK

Fit Width and Height

Click the Fit the Image Width button to make the studies as wide as the window.

 

Use the Fit the Image Height button  to make the first page of the studies as tall as the window area of the study display.

 

The image will not become smaller than half the width of the window (Zoom of 50).

BACK

Zoom

Use the Zoom track bar  to zoom in or out. A zoom of 100 will fill the width of the window.

BACK

Resetting to the Top Left Corner

The Resetting to the Top Left Corner button moves all studies to the top left corner if they have been scrolled away from that position.

BACK

Paging Controls

Some studies contain multiple pages.  There are several controls that allow you to go to certain pages.  These paging controls are only enabled if only one study is visible and that study has multiple pages.

The First Page button  takes you to the top of the first page in the study.  

The Previous Page button  takes you to the page before the one that is currently visible.  

The Next Page button takes you to the next page after the one that is currently visible.  

The Last Page button takes you to the last page of the study.

 

As you use the paging controls or the vertical scroll bar, the Current page text box updates to show you were you are within the study.  Changing that text box to another number will move the page to the page specified.

BACK

Locked Scrolling

Clicking the Lock Scrolling button will lock multiple tests together.  Scrolling one of the studies will also to scroll the others by the same amount.

BACK

Viewing Studies

You can use the scroll bars around each study to view different areas of each study.  If the study is small enough to fit within the study’s area, the corresponding scroll bar may not be visible.

BACK

Study Controls

Using the study controls, you can move through the list of studies without opening the studies dropdown.  

 

The First Page button takes you to the top of the first study in the list.  

The Previous Page button takes you to the study after the one you are currently viewing.

The Next Page button takes you to the study after the one you are currently viewing.   

The Last Page button takes you to the last study in the list.

BACK

Selecting Studies

If the study dropdown is not open, click the Select Studies to Display button .

 

Click on the study you would like to view.  If you would like to view more than one study simultaneously, hold CTRL and select additional studies.  To view more than one study that are in sequential order, select the first one, hold SHIFT, and select the bottom one.  To close the study list dropdown, click the Select Studies to Display button .

 

If you want to only view a single study, you can double click on the study in the study list dropdown which will select that test and close the study list dropdown.

BACK

Sorting Studies

Studies may be sorted by clicking on the head of the column in the study dropdown.  Clicking the column again will resort that column in reverse order.

BACK

Unconfirmed EKGs

EKGs that have been automatically interpreted by the MUSE server but have not been confirmed by a cardiologist have an UNCONFIRMED overlay at the top of the EKG report.

 

The study list also contains a column indicating whether an EKG has been confirmed or not.

 

In addition, the overlay text indicates when an EKG is unconfirmed.

BACK

Print Settings

Clicking on the Print Settings button   brings up the settings window.  This allows you to change the printing options.

 

Some printers have trouble printing the grid accurately.  Choosing Print grid with dotted lines should fix this problem.  This setting is saved so the next time you launch the application it will remember this setting.

BACK

 Printing

To print, click the Print button .  This will print all pages of all studies on the screen.  If the grid is on, it will also print the grid.  See Print Settings if the grid does not print correctly on your printer.

BACK

Refreshing the servers

The EKG Display client remembers if a MUSE server is unreachable and does not attempt to connect to it again during this session.  This prevents a 30 second timeout wait.
