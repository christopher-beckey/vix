Patient EKG Tests window

On the tool bar of the VistA MUSE EKG Display window is the Patient Tests Window button.

<Click> the button and the 'Patient Tests dropdown list' will be hidden and the window (shown below) will be displayed.

 

The window can be resized to see more tests. Select an item and that test will be displayed.

 

Close the window and the Patient Tests drop down list will be redisplayed
