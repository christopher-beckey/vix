Starting the MUSE EKG Display

The VistA Imaging MUSE EKG Display interface is started from the VistA Imaging Display main window.

From the VistA Imaging Display main window, selected a patient, then either click on the MUSE EKG button on the Imaging System window tool bar.

 

or select the View | MUSE EKG window list menu option and the VistA Imaging MUSE EKG Display window will be started.
