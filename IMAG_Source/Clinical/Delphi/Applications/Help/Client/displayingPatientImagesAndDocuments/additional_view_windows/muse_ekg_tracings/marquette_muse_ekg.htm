Marquette MUSE EKG

The VistA Imaging team has developed an interface to the Marquette MUSE Database.  The Interface uses an API (Application Programming Interface) developed by Marquette that allows Applications to access patient data from the MUSE Database.

 

The functions provided are

  • Look Up patients in the MUSE database by:

  • entering partial or full Name

  • entering patient's full SSN

 

Display an EKG by selecting  from a list of patient EKG's

Select a certain type of test to display, i.e. only Resting

 

Manipulate the EKG displayed by

  • Zooming IN or OUT

  • Display Grid lines  ON / OFF

  • Fit Image to width

  • Fit Image to height

 

Print Image

  • Change the % that the Image is increased / decreased when using Zoom functions

  • Display the Next / Previous page of an EKG Image.

  • Settings are saved when the VistA Imaging MUSE EKG window is closed.

 

Caution: The possibility of accessing two separate Patient Databases is possible.

The VistA System Patient Record, and the Marquette MUSE Database. Caution is emphasized because there will be times when a patient in one database is not in the other. When this happens, contact the system manager.
