Video File Options window

The Video File Options window allows the selection of Video Viewers.

 

If 'Use Alternate viewer' is selected, then video files will be opened in whatever viewer is associated through Windows for files with the same extension. By default, this is the Windows Media Player, MPlayer2.exe.

 

 

Select 'Use Imaging Video Player' to use the Video Player window supplied with VistA Imaging. Check 'Play Video File when selected' to automatically start the video when it is loaded in the Imaging Video Player window.

 

Note: In Version 3.0 only video files with the AVI extension are able to be captured to the patient record. Version 3.0.8 adds support for MPG files.
