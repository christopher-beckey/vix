MUSE patient lookup

Click the 'Patient' button on the MUSE EKG Display window to open the MUSE Patient Lookup window.

 

Type a Full or Partial patient name in the 'Name' field and press <Enter>. The first 20 Patient names that follow the text will be displayed.

 

Type a Full SSN into the 'SSN' field and press <Enter>, If the SSN is found in the MUSE Database, the tests for that patient will be listed in the patient tests drop down list box on the MUSE EKG Display window.

 

If the SSN isn't found, a message window will be displayed.

Select a patient by using the mouse and clicking on a patient entry. or using the arrow keys to move up or down the list.

 

When the patient is highlighted, click the OK button.

 

Clicking the Cancel button will close the patient look up window and position the cursor back in the MUSE EKG Display window.

 

Clicking on More will add the next 20 patients to the patient listing.

 

<double-click> on an entry will also select the patient.

 

Caution: The possibility of accessing two separate Patient Databases is possible.

 

The VistA System Patient Record,  and the Marquette MUSE Database. Caution is emphasized because there will be times when a patient in one database is not in the other. When this happens, contact the system manager.

 

In the VistA Imaging System main window:

When a patient is selected from the VistA Imaging System main window. The VistA System Patient Record is being accessed. <Click> on the MUSE EKG Button or select the MUSE EKG Display menu option, and the SSN of the currently selected VistA System patient is used to do a look up into the Marquette MUSE Database. The VistA Imaging MUSE EKG Display window is opened and if the SSN is found in the MUSE Database then the 14 most recent tests are listed in the patient test drop down list box on the MUSE EKG Display window.

 

If the SSN isn't found in the MUSE Database a message window will display a message stating that no matching SSN was found.

 

In the VistA Imaging : MUSE EKG Display window:

<Click> on the 'Patient' button. The MUSE Patient Look Up window is opened.  Enter a full or partial patient name in the Patient edit box and press <Enter>. A look up into the MUSE Database is started.  The first name matching the text entered in the Patient edit box is listed, followed by the next 19 patient names.

 

When an SSN is entered in the 'SSN' field and <Enter> is pressed, a look up into the MUSE Database for a patient with that SSN is started. If the SSN is found, the 14 most recent tests are listed in the patient test drop down list box on the MUSE EKG Display window. If the SSN isn't found in the MUSE Database a message window will display a message stating that no matching SSN was found.
