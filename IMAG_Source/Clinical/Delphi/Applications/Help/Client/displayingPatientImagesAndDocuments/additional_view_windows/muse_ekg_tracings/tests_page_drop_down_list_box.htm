Tests Page drop down list box

The Tests Page drop down list box displays all pages for the current test.

The count of total pages for the current test is displayed to the right of the drop down list box.

 

Note: If the selected test has only one page, the Tests Page drop down list box will not be displayed.

 

Note: When focus is on the Tests Page drop down list box, use the <up> and <down> 'arrow keys' to display the previous or next Page.
