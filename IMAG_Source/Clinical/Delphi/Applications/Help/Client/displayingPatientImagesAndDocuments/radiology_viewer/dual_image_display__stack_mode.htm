Dual image display: Stack Mode

Also see: Stack View Options

 

Two radiology images can be displayed at once.

  • First click the left mouse button on a radiology abstract image in the Group Window.
    • The image will be displayed in the Radiology Viewer window.
  • Next click the right mouse button on a radiology abstract image in the Group Window,
    • and select the shortcut menu option 'Open in 2nd Radiology window'

 

 

A yellow description bar beneath an image indicates that it is the selected image.

  • Only one of the radiology images can be "selected" at a time.
  • To select an image, left click on it with the mouse.

 

Image manipulation operations (performed using the tool bar or menus) can be applied to:

  • either the "selected" image
  • or to "all" images.

This choice is controlled by clicking the "Apply to All" button on the control bar.

  • If the "Apply to All" button is depressed,
    • the image manipulation operations will be applied to all images.
  • Conversely, if the 'Apply to All' button is not depressed,
    • the image manipulation operations will be applied only to the "selected" image.
