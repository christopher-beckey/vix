Radiology viewer - ViewSettings

There are four different options for the radiology viewer window.

    • One image can be shown,
    • two images can be shown side by side,
    • or there can be a layout of:
      • either 6 images (using the 3x2 Layout),
      • or 12 images (using the 4x3 Layout).
  • To choose the radiology viewer settings, click on ViewSettings in the menu bar, then click on the appropriate option.

 
  • The stack settings allow a different study (or the same study) to be shown in each window.
  • The user can page forward and backward through the stack(s), viewing each image like you would cards in a deck.
  • Two stacks can be paged simultaneously by selecting the Apply To All button.
  • When either of the Layout settings are selected, only images from one study can be displayed together.
  • When the user clicks on an image in the Group Window, that image is shown in the first position. Sequential images in the study are displayed in the remaining positions.
  • The user may page through the images one-at-a-time, a page-at-a-time, or by moving to the beginning or end of the study.
