Stack View Options

'Page Together' allows the two stacks to be paged together when moving through the images in a study.

When two studies are paged in a Stack View, there are three methods for determining the window and level settings of the images.

 

If 'Page with Different Settings' option is selected,

  • then the images in each stack will use their own window level settings.
  • The window, level, and zoom settings will be the same for each image in the stack.
  • This can be used to view a CT scan with two different CT Preset values.

 

For example, the same Thorax CT study is loaded into both stacks.

  • Then the image in the first stack is assigned a CT Preset, such as lung.
  • The image in the second stack is assigned a different CT Preset, such as mediastinum.
  • The 'Apply To All' button is clicked so it is in the depressed position.
    • Now, when the paging buttons are clicked,
      • both stacks advance simultaneously,
      • and hold their assigned Preset values.

 

If 'Page with Same Settings' options is selected,

  • then all images in both stacks will use the same window level settings.
    • This can be used to compare two CT studies performed at different times.
  • Again, the 'Apply To All' button should be clicked so it is in the depressed position.
  • When the paging buttons are clicked,
    • both stacks advance simultaneously, with the same window, level and zoom values.

 

If the 'Page with Image Settings' option is selected,

  • then all images in both stacks will set the window and level values
    • according to the selected "Image Settings" option.
  • This means that each image loaded in either stack
    • will use the device settings or the histogram settings,
      • whichever has been selected under the Image Settings options.
