To view a radiology image or group of images, select the study from the Abstract window, Group Abstracts window, or Image listing window by clicking on it. Radiology images are generally part of a study group of images and the abstracts are normally displayed in the image Group Abstracts window.

 

To view a radiology image, select the study from the Abstract window or Image Listing window by clicking on it. If there is only one image in the study, it will be displayed in the Radiology Viewer window. If there are more than one image in the study, a Group window will open showing abstracts for the images in the study. You may page through the group to select images of interest. Click on the image you wish to view. The image will be displayed in the Viewer window. If a Layout view has been selected using the ViewSettings options, then several consecutive images will be displayed.

 

When displaying studies that contain both color and black & white images, be sure that your settings are as follows:

 

  • For Stack View:    Page with Image Settings

  • For Layout View:  Use Individual Image Settings

 

This will cause the color images to display as color, and the black and white images to display correctly as black and white.

 

 

The Radiology viewer is brand new and contains all of the functionality of the old viewer plus new functions.  

  • It supports more image types by using the new ImageGear toolkit and provides improved options for image manipulation.  

  • The user can use the scroll wheel on their mouse to move between images in a study.  

  • The user can double-click on an image in the stack view to view the study in layout mode

    • and can double click on an image in the layout mode to view the image in a stack.

 

There will be a caption below each radiology image displayed. The caption will indicate the type of study, the study description, and image information including pixel depth (8, 10, or 12 bits), image width and image height.

 

VistA Imaging will display the size of the image at the bottom of the window. Typical radiology image sizes are as follows:
