  •   Window/Level Scroll bars: The window (contrast) and level (brightness) values may be changed by moving the scrollbars.

  •   Use the mouse to drag the scrollbar to a new position.

  The Zoom scrollbar may be used to select a zoom factor.

  To pan the image, click on the Pan button on the toolbar.

  • Place the mouse cursor over the image and press down on the left mouse button.
  • Drag the mouse to move the image within the radiology window and release the mouse button when the image is in the desited position.

Note: Panning will work when the image is zoomed and scrollbars appear on the edge(s) of the image.

When using this feature,

  • first click on the Pan Window button,
  • then use the mouse to click on the image.

 

A Pan Window will open in the upper left corner of the screen.

This Pan Window box will display a full sized version of the image with the zoomed portion highlighted.

  • The scrollbar at the bottom of the box can be used to adjust the amount of magnification.

  • The mouse can be dragged around the image to move the highlighted area.

  • Click on the Magnify Window for more information.

 

  ROI (Region of Interest) Window/Level:

  • This control allows adjusting the brightness and contrast of the image so that a selected region will be viewed optimally.

  • Click on the AutoWin/Lev button on the tool bar. (the button will now appear depressed). Now, move the mouse to the upper right corner of the region to be selected and press the left mouse button down.

  • Drag the mouse to the lower left corner of the region to be selected.

  • Release the mouse button.

  • The window and level of the entire image will be modified automatically so the selected region will use the entire gray scale range.

  • Click the Copy button to copy the image to the Windows clipboard.

  •   The Report button may be clicked to display the short description and the VistA radiology report associated with this image.

  •   Click on the Print button to print the image to a network printer.

  • The page header will contain the patient's name and image description.

  •   The Apply to All button enables the user to to make the same changes to all images simultaneously.

  •   When the button is clicked again, the images can be manipulated individually.
