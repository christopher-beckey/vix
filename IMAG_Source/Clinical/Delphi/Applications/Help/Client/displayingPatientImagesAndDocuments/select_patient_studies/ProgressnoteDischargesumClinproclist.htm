Progress Note, Discharge Summaries, Clinical Procedures list

Progress notes, Discharge Summaries and Clinical Procedures are listed in this window. They are displayed by the menu on the VistA Imaging Display main window:

  • menu options

 

Progress Notes: click View, then select Progress Notes.

 

 

 

Discharge Summaries: click Reports, then select Discharge Summary.

 

Clinical Procedures: click View, then select Clinical Procedures.

 

The Number of entries listed, and the type of entries will be displayed directly above the list. If the menu option 'Options | Preview Selection' is checked, then whenever a new entry is selected, the report will display in the bottom pane of the window.

 

Sorting the columns:

Click on a column header to sort the list by that column. Click on the same column to reverse the order of the sort.

 

 

Preview Selection

Check this to display the report in the bottom pane of the window whenever a list entry is selected.

Refresh the list

Retrieve the latest list of Notes or Discharge Summaries from VistA

Select Author

Select the 'Author' to list images for a specific author.

Note: The list can be sorted by Author, by clicking on the 'Author' column header.

List Count

Select how many of the most recent Notes to display.

Font

Change the font of the list.
