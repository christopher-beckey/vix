Image Filters

Also see: Image Filter Add/Edit window, Custom filter (used for testing),

Public, Private and Default filters

 

Image filters allow the user to view subsets of patient images.

This feature speeds:

  • the display of image lists

  • the retrieval of the desired images

 

Filters contain image 'index fields':

Class

Administrative, Clinical

Package

MED, SUR, RAD, CP, etc

Type

  • Administrative: Means Test, Forms, Advanced Directives

  • Clinical: Diagrams, Image

 

Specialty/Subspecialty

Medicine. Radiology, Ophthalmology, etc.

Procedure/Event

ABG, Endoscopy, Cardiology, Radiology, etc

Origin

VA, Non-VA

Date Range

Individual date or period of time

 

 

BEFORE

 

 

When VistA Imaging Display is started the user's Default Filter will be displayed in the message bar.

  • When a user ‘Saves Settings’ the current filter (unless it is a temporary ‘Custom’filter) is saved as the default filter for that user.

AFTER

  • After selecting a Patient the message bar will contain the total image count for the patient and the number of images that match the Filter in use.

  • the message shown above means that there are 12 images for this patient and the current Filter has selected all 12 for viewing

 

 

There are 3 types of Filters:

  • Public filters are predefined and cannot be changed.

  • Private filters are user defined and can be saved, modified and deleted.

  • Custom filters are temporary,

    • expire when the user ends the current session.

    • can be used to test a filter prior to saving it as a Private filter

  • for additional details see => Public,Private and Default Filters

 

The Image Filter dialog is used to define/modify/delete filters.

It is accessed by:

  • clicking the  

    • in the main Imaging Display window or

    • Image List window

OR

  • via the Options menu in the main Imaging Display or Image Listing window.
