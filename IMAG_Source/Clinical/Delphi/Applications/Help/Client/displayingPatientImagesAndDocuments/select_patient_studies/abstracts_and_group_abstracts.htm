Abstracts and Group Abstracts

Also see: Abstracts and Group Abstracts-shortcut menu, Generic bitmaps used for abstracts, Image proxies, Image Information dialog

 

The Abstracts and Group Abstracts windows display smaller versions, called abstracts, of each single image and image group on file for the selected patient.

 

To display the shortcut menu, position the cursor over the window and press the right mouse button, or press <Ctrl-Tab>

 

The user can now select either "top" or "left" for the position this toolbar.

 

Abstracts are displayed in the above window.

 

Image Dates are now displayed in the Hint:

  • in the Abstract, Group Abstract, Full Resolution and Abstract preview of the Image List window

 

When the window is resized or scrolled, the dark gray areas are replaced automatically by image abstracts.

 

This function replaces the need for Image Proxies (implemented in previous releases of VistA Imaging Display).

  • The proxies appear below as white rectangles.

  • The number of proxies appearing was based on the size of the window and the Image Viewer Settings dialog.

 

Click on an abstract to see the full-resolution version of the image.

  • If the abstract actually represents a group of images, the Image Group window will be opened

Note: An Image Group is represented by the image description having a black background.

 

Image proxies are a feature available in the Abstracts, Group Abstracts, and Full Resolution View windows.

  • Proxies are displayed as a white rectangle.

 

The Title of the Abstracts and Group Abstracts windows displayed the number of abstracts currently loaded.

  • Use the Scroll bar to view the loaded images.

  • Use the page buttons on the Toolbar to view the next or previous page of abstracts.

 

Features of the Abstracts and Group Abstracts windows:

  • The ability to scroll to any abstract in the list is included in the windows themselves. Horizontal and vertical scroll bars actively reflect the availability of additional abstracts.

  • When the Abstracts or Group Abstracts windows are opened, all available images are loaded.  Scroll bars can be used to view any additional abstracts.

  • All abstracts are loaded, the title bar in each window reflects the total number of images.

 

The following are options of the shortcut menu (displayed when an abstract is right-clicked).

 

Open

Opens the selected abstract into the appropriate viewer (same as left-clicking the image).

Image Information

An image information window, available to all users, will be opened when this option is clicked.

Image Information Advanced

This is a function available to users who have the MAG SYSTEM KEY.
