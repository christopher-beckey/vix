Viewing

VistA Imaging provides the ability to view any report associated with an image.  Images can be associated with the following reports:

 

  • Medicine procedure reports

  • Surgery case reports

  • Anatomic pathology reports including autopsy reports

  • Radiology reports

  • Progress notes

 

To view the associated procedure report for an image

  • Click the report button from the toolbar or select the shortcut menu option 'Image Report'. The report will be displayed in the Image Report Window.

    • All windows in VistA Imaging that display images, abstracts, list of images or list of patient reports will have the report button on the toolbar and/or have the 'Image Report' option in the shortcut menu for the window. (<right-click> the mouse on the image or selected list entry to display the shortcut menu).

  • VistA Imaging also provides the ability to view certain Patient Reports. Patient reports include:

    • Health Summary

    • Patient Profile

    • Discharge Summaries

 

These reports are available from the 'Reports' menu on the VistA Imaging display window. Select a menu option to have the report display in the Image Report Window.
