vista imaging - clinical display

Overview

Patch 72 upgrades the imaging component used within the Clinical Display client to AccuSoft ImageGear 15.4 (currently). This component is now used throughout the application for viewing of all images. By using this new component we have fiexed several problems we had with the previous version of AccuSoft ImageGear (IG99) including the odd-columns and viewing embedded thumbnails.

 

Included in this patch is a brand new Radiology Viewer that makes use of some of the features available in this new image toolkit. This new viewer is based on the functionality of the old Radiology Viewer but has a look and feel that matches more closely to the Full Resolution viewer. The goal is to make it easier for users to access buttons and functions.

 

There are also new components needed to communicate with the DoD and view DoD images. These components allow images and studies from the DoD to appear as VA sites. If a patient has been seen at the DoD, the DoD site will appear on the Remote Image View toolbar as it would on a VA site. Communication with the DoD is done through the VistA Imaging Exchange (ViX) server (Patch 83). A site can only view DoD images if it has a ViX installed.

 

Features Description

PDF Viewing

In Patch 72, PDF documents are now viewed using the Full Resolution viewer instead of using the Adobe Reader.

  • This allows users to compare PDF documents side by side with each other and other images in the patient records.

 

Note: A feature to be added will allow users to decide if they want to use the Full Resolution viewer  or the Adobe Reader to display PDF documents. This option will be stored as a user preference.

 

DoD Images

The Patch 72 Display client can view DoD images by communicating with a local VistA Imaging Exchange (ViX) server. If a site does not have a ViX server it will be unable to view DoD images.

 

A DoD button will appear on the main Display client window in the Remote Image View toolbar for patients who have been seen at the DoD.

 

When the user chooses to connect to the DoD:

  • the Display client communicates with the local ViX server to retrieve DoD studies and images.

  • DoD studies are displayed for the user in the same format as VA images (in the image list window and the abstract window).  

Studies are sorted by procedure date with the most recent studies appearing first.

 

 

Users can use the same tools to view DoD images as they would use to view VA images including all of the functionality of the Radiology Viewer.

 

Radiology Viewer

 

The Radiology viewer is brand new and contains all of the functionality of the old viewer plus new functions.  It supports more image types by using the new ImageGear toolkit and provides improved options for image manipulation.  

    • The user can:

      • turn the scroll wheel on their mouse to move between images in a study.

      • double-click on an image in the stack view to view the study in layout mode,

      • and can double-click on an image in the layout mode to view the image in a stack.

 

Feature Enhancements

The Radiology viewer has several new features.

 

The user can now use the scroll wheel on their mouse to quickly move between images in the stack and layout views.  This allows images to move very quickly so the user can see the difference between images.

 

  • Double-clicking on an image in the:

    • layout view makes the image appear in the stack view at full size.  

    • stack view makes the study switch back to layout view.

 

Measurements

When making ruler and angle measurements on the image,

  • if the user right clicks anywhere on the image:

    • the selected tool will change to the ruler/angle pointer to allow the user to select and modify measurements.  

    • If a measurement is selected the user can press the delete button to delete the selected measurement(s).  

    • If the user right clicks on a measurement a dialog will appear that allows the user to change properties of the mark:

      • including color, line width, and measurement scale.

Note: Measurements are not kept when changing between images.  Once a new image is displayed, the measurements from the image are removed.

Radiology Viewer - Image Settings Options

When an image is loaded into the Radiology Viewer, there are two methods for determining its initial window and level settings.

In some cases, the Radiology device that created the image indicates appropriate Window and Level settings in the associated image data (also called, the image header).

  • By selecting the "USE DEVICE WINDOW LEVEL IF AVAILABLE" option,
    • any settings provided by the device will be used.
    • If none are provided, the histogram option will be used.
    • This will generally provide a good image display, unless the device does not send proper window and level values.

If the Histogram option is selected, then window and level values are selected based on the gray levels detected in the image. This will generally provide a resonable image display.

 

 

 

Radiology Viewer - ViewSettings

There are four different options for the Radiology Viewer window.

  • One image can be shown,
  • Two images can be shown side by side,
  • There can be a layout of 6 images (using the 3 x 2 layout)
  • 12 images (using the 4 x 3 layout)

To choose the Radiology Viewer settings,

  • click on ViewSettings in the menu bar,
  • then click on the appropriate option.

 

 

The stack settings allow a different study (or the same study) to be shown in each window.

  • The user can page forward and backward through the stack(s), viewing each image like you would cards in a deck. Two stacks can be paged simultaneously by selecting the Apply To All button.
  • When either of the Layout settings are selected, only images from one study can be displayed together.
    • When the user clicks on an image in the Group Window, that image is shown in the first position.
    • Sequential images in the study are displayed in the remaining positions.
    •   The user may page through the images one-at-a-time, a page-at-a-time,
      • or by moving to the beginning or end of the study.

 

Radiology viewer - layout view options

When a study is paged in a Layout View, there are two methods for determining it's initial window and level settings.

  • All images can be displayed with the settings selected for the first window.
  • This allows users to select their own window and level settings, either by using the window/level tools or by selecting a CT preset value. Settings may be changed when needed.
  • Image may be loaded using the window/level settings determined by the selection of "image settings" in the option menu.
  • This can be either the device window/level or the histogram window/level.

     

 

Dual Image Display: Stack Mode

Also see: Stack View Options

Two radiology images can be displayed at once.

  • First click the left mouse button on a Radiology abstract image in the Group Window.
  • The image will be displayed in the Radiology Viewer window.
  • Next, click the right mouse button on a Radiology abstract image in the Group Window, and select the shortcut menu option "Open in in second Radiology window".

A yellow description bar beneath an image indicates that it is the selected image.

  • Only one of the Radiology images can be selected at a time.
  • To select an image, left-click on it with the mouse.

Image manipulation operations (performed using the tool bar or menus) can be applied to either the selected image or to all images.

  • This choice is controlled by clicking the "Apply to All" button on the control bar.
  • If the "Apply to All" button is depressed, the image manipulation operations will be applied to all images.

Conversely, if the "Apply to All" button is not depressed,

  • the image manipulation operations will be applied only to the selected image.

 

Stack View Options

"Page Together" allows the two stacks to be paged together when moving through the images in a study.

  • When two studies are paged in a Stack View, there are 3 methods for determining the window and level settings of the images.
  • If "Page with Different Settings" options is selected, then the images in each stack will use their own window/level settings.
    • The window, level, and zoom settings will be the same for each image in the stack. This can be used to view a CT Scan with 2 different CT Preset values.
    • For example, the same Thorax CT study is loaded into both stacks.
      • then the image in the first stack is assigned a CT Preset, such as "lung".
      • The image in the second stack is assigned a different CT Preset, such as mediastinum.
      • The "Apply to All" button is clicked, so it is in the depressed position.
      • Now, when the paging buttons are clicked, both stacks advance simultaneously, and hold their assigned Preset values.
  • If "Page with Same Settings" options is selected, then all images in both stacks will use the same window/level settings.
    • This can be used to compare two CT studies performed at different times.
    • Again, the "Apply to All" button should be clicked so it is in the depressed position.
    • When the paging buttons are clicked, both stacks advance simultaneously, with the same window, level, and zoom values.
    • For "Page with Same Settings" to be enabled, "Page Together" must also be checked.
  • If the "Page with Image Settings" option is selected, then all images in both stack will set the window/level values according to the selected "Image Settings" option.
    • This means that each image loaded in either stack will use the device settings or the histogram settings, whichever has been selected under the "Image Setting"s options.

 

 

Measuring

Measurements can now be made directly on the image being viewed. To take measurements press the   button on the toolbar to activate the Ruler tool.

Note: For certain images the button might not be enabled. This is because the information necessary to accurately calculate the measurement is not available in the image information.

 

Use the mouse to place the cursor over one end of the object to be measured.

  • Click down with the left mouse button.
  • Hold the left mouse button down while moving the cursor to the other end of the object to be measured.
  • Release the mouse button.
  • A line will appear to indicate the length measurement that was made.
  • The measurement will appear on one side of the ruler line.

 

Measuring Angles

If angle is selected, the user will trace the angle.

  • Use the mouse to place the cursor at the point of the vertex of the of the angle to be measured.
  • Click down and and up with the left mouse button.
  • Move the cursors to end of one of the line segments and click down and up with the left mouse button.
  • Place the cursor over the end of the second line segment of the angle to be measured,
    • and click the left mouse button again.
  • A line will appear, to indicate the angle measurement that was made.
  • The measurement will appear near the vertex of the angle.
  • Multiple measurements may be made; the results of each will be displayed.
  • To clear the lines from the image,
    • select the 'Tools' menu,
    • then select 'Ruler/Angle Tool'
    • and select 'Clear All Measurements'.
  • The user can also use the button to select measurements and move them or delete them by pressing the Delete key or from the 'Tools' menu,
    • select 'Ruler/Angle Tool' and select 'Delete Selected Measurements'.
  • Note: Measurements are not kept when changing between images.
    • Once a new image is displayed, the measurements from the image are removed.

 

Measurement Options

From the 'Tools' menu,

  • select 'Ruler/Angle Tool'
  • and select 'Measurement Options' to display the Measurement Options dialog.

This allows the user to change the line width, the measurement units and measurement color.

Pressing the 'Apply' button will apply the changes and keep the dialog open to allow further modifications.

Pressing the 'OK' button will apply the changes and close the dialog.

Pressing the 'Cancel' button will cancel any changes made.

These settings are saved as user preferences for future Imaging sessions.
