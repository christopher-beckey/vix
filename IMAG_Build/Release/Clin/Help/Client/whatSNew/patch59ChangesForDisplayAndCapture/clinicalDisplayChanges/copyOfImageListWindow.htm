Click this graphic  to open the Image Filter dialog.

  • The Image List Window lists the available images for the patient.

  • Clicking on a Column Header results in sorting the list by that column.

 

The # column indicates the number of images attached to that procedure.

  • The widths of the columns can be changed on this window by selecting the column dividers and dragging them to the left or right.

 

To display the shortcut menu, press the right mouse button or press <Ctrl-Tab>.

 

See Menu Options for an explanation of using the imaging list window to Browse patient Images.

  • The modified Image List window is shown below:

 

 

Features:

  • The number of images is available in the '# Images' column.

  • The image list includes columns for the index fields introduced in Patch 17: Package, Class, Type, Proc/Event, and Spec/SubSpec.

  • A menu bar provides access to all features in the Image List window.

  • The column widths in the image list are saved with User Preferences.

  • The Proc Dt (procedure date) and Cap Dt (date/time image saved) columns now display the time of day, if time of day data is available.

  • The 'Note Title' column has been added. If the Image is attached to a progress note, this column displays the title of the note.
