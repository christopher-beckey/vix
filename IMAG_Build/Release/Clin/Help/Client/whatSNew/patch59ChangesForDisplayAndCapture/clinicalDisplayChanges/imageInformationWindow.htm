Image Information Window

The Image Information window now reports the time relationship between the attachment of an image and the status of the image’s associated progress note.

 

For signed or electronically filed (see note 1) progress notes, one of the following lines may be displayed:

 

Image was attached Same Day as Note was Signed (see note 2)

Image was attached Same Day as Note was Electronically Filed (see note 2)

Image was attached After the Note was Signed

Image was attached After Note was Electronically Filed

Image was attached Before the Note was Signed

Image was attached Before Note was Electronically Filed

 

For unsigned progress notes, the following is displayed:

Image is attached to an Unsigned Note

 

For notes with addenda (see note 3), the following is displayed:

Image is attached to an Addendum

 

Notes:

1. The “electronically filed” status replaces the “administrative closure: scanned document” status. This change is described in more detail in the Sign or Electronically File Progress Notes section.

 

2. If the time of the event is not specified when the image captured, but if the dates of the note and image, when compared, are equal, then Image was attached Same Day as...  will be displayed.

 

3. An “addendum” is a new, optional way to attach more images to a note after the note has been signed. Addenda are described more detail in the New “Make Addendum” Option for Progress Notes section.
