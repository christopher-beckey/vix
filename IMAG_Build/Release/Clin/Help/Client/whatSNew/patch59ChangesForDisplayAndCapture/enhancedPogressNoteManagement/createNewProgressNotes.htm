Create New Progress Notes

Users can now create a new progress note in Capture, reducing the need to switch back and forth from CPRS. To create a new note, the user clicks Select Progress Note in the main Capture window, and then uses the new Create New Note tab in the Progress Notes dialog.

 

 

(1) The Author box is pre-populated with the current user’s name. Users can enter the name of a different user as well.

 

(2) The Visit Location box needs to be populated by the user.

 

For the Author Box and the Visit Location Box, entering the first few characters will display a list of all entries that match the characters typed.  The User does not have to Press Enter.

 

(3) The Note Status area allows the user to indicate what the status of the note should be set to once the image capture is complete. The ability to set a note’s status is new for Patch 59, and is described in more detail in Sign or Electronically File Progress Notes.

 

(4) The Enter Text box can be used to enter text for the note. Boilerplate text, if defined, may be present already. (Templates, however are not implemented. To work with a template, the note must be created using CPRS.)

 

(5) The note title area can be populated by selecting a note from the scroll list, or by entering the first few characters of the note title (which causes a clickable list of matching note titles to display).
