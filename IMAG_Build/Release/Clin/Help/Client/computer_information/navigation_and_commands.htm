Navigation and Commands

The following conventions are used in this manual to describe the navigation and commands users perform in VistA Imaging.

 

 

User Keyboard Entry

<Enter>

Enter key

<Shift>

Shift key

<ESC>

Escapte key

<Num Lock>

Top left key on the numeric keypad (above the 7), may also be labeled Numeric Lock; this makes any keypad key activate the number shown on its surface; it is the equivalent of a SHIFT LOCK for alphabetic keys.

Menu | Option |...

This convention is used to explain a sequence of selections through a set of menus and/or options.

 

For example, "Select 'Options|View Preferences' from the VistA Imaging System window" would mean that: from the VistA Imaging System window, select the 'Options' menu, then select the 'View Preferences' option

Text Box

These are boxes into which keyboard input, such as the Access Code text box on the VistA Sign-On screen.

 

Radio Button

Radio buttons and check boxes serve to enable or disable general options or preferences, such as the List Images in Reverse Order check box on the View Preferences window.

Check Box

 

(see Radio Button - above)

<Ctrl-TAB>

Most windows in VistA Imaging have shortcut menus, such as the Image Listing window, that are opened by pressing the right mouse button <right-click> or by holding the <Ctrl> button down and pressing the <Tab> key.

 

Use the arrow buttons to highlight an option on the shortcut menu, and press the <Enter> key to select the option.
