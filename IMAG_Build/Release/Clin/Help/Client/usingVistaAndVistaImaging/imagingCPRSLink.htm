Imaging-CPRS Link

Both VistA Imaging Display and Capture can be started from the CPRS menu option 'Tools'.

 

If this option isn't available on the CPRS menu, see the System Manager.

 

Overview

When VistA Imaging is started from CPRS option, it looks slightly different.

 

  • The patient edit box, is disabled. Patient look ups are not available.

  • The 'Login', 'Logout' and 'Remote Login' menu options are disabled.

 

 

These changes were made so that Imaging and CPRS stay connected to the same Site and are viewing the same patient.

 

Note: Similar changes happen to Imaging Capture. Patient look up, 'Login', 'Logout' and 'Remote Login' menu options are disabled.

 

 

Note: A menu option ('CPRS Link Options') is available on the 'Options' menu in both Imaging Display and Imaging Capture when VistA Imaging is started from CPRS.

 

The  'CPRS Link Options' will Open the 'Imaging <-> CPRS Link' window.

 

The CheckBox, 'Synchronize with CPRS Patient /Note/Exam', will be checked when Imaging is started from the CPRS Tools menu. Whenever a patient changes in CPRS, Imaging will automatically change to the new patient.

 

Whenever a Progress Note, Radiology Exam are selected in CPRS, any associated images will be displayed in the Imaging Group Abstract window.

 

When Synchronized with CPRS, Patient look up, Login, Logout, Remote Login are disabled in the Imaging Menus.

 

To enable the patient edit box, the 'Login', 'Logout' and 'Remote Login ' options, Uncheck the Synchronize check box. This will enable the Imaging user to connect to a different site and look up a different patient than is being viewed in CPRS.

 

Note: To synchronize with CPRS after the Link is broken, VistA Imaging must be closed, then started again from the CPRS Tools menu.
