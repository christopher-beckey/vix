Images Associated with Clinical Procedures

The VistA Imaging capture software allows images or scanned documents to be captured and associated with Clinical Procedures. The images that were previously associated with the Medicine package (endoscopy, hematology, pulmonary, GI, cardiology, rheumatology, ophthalmology, etc) may be associated with the Clinical Procedures package using VistA Imaging.

 

The user must have a Clinical Procedures capture key.

 

Using the configuration option, the Association should be set to Clinical Procedures. The correct Source and Study/Single save options should be selected.

 

 

The user will identify the patient as usual. Next the user should click on the "Select Clinical Procedure" Button.

 

A list of "Clinical Procedures Requests" for the selected patient will appear.
