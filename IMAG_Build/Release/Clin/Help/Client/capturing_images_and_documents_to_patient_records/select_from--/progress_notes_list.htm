vista imaging capture

Progress Notes list

See also: Select Note, Create New Note

From the VistA Imaging System - Capture window click the  'Select Progress Note' button.

 

Select the Note to be associated with the captured image, and click on 'OK'.

 

Select Note

 

Options:

Create Note

Write a new Note.

Make Addendum

Add an Addendum to a Note.

Sign Note

Sign the completed Note.

Clear Text

Remove the text in the Note.

Refresh

Gets the latest list from VistA.

Preview Selection

Shows/Hides the Progress Notes viewer.

Fit Columns to Text

Adjust the size of the columns to fit the amount of text.

Fit Columns to Window

Adjust the column size to fit the window.

Select Columns

Select a column of interest.

List Options

Display the options provided.

Font

Change the Window font.

Reset Panel Size

Restore the section to its original size.

 

View:

Show Related Notes/Addendums

 

My Signed Notes

 

Signed Notes (All)

Lists most recent notes.

UnSigned Notes

Lists most recent Unsigned notes

UnCosigned Notes

Lists most recent UnCosigned notes.

Signed Notes by Author

Lists most recent notes signed by the selected Author.

 

Note: When 'Signed Notes by Author" is selected, a selection window will open.

  • Select an author and the list will be limited to notes of that author.
