Tabbing Sequence

Tabbing sequence is the pre-defined order for the controls in a window. Pressing <TAB>, moves the input focus to the next control in the tabbing sequence. All controls in a window aren't necessarily in the window's tabbing sequence. Controls that are disabled are skipped when they are the next control in the tabbing sequence.

 

Note: VistA Imaging provides an option to change the placement of the edit fields and image function buttons on the Main Capture window by selecting the menu option 'Option|Field 'Tabbing' Sequence'

 

When the order of the controls is changed the tabbing sequence is also changed. The new tabbing sequence starts with the control at the top left and proceeds in a left to right, top to bottom direction.
