Configuration Settings displayed

The image capture settings can be changed by:

  • the Imaging Workstation configuration window,

  • the Imaging Workstation: Capture configurations list window,

  • clicking a Configuration button, located just below the menus, (Source, Format, Association, Saving, Mode, Other),

  • or

    • clicking a user-created configuration button.

 

(top of the VistA Imaging Capture window)

 

Near the top of the VistA Imaging Capture window, under the menus,

  • In black are the Configuration buttons.

    • input Source, image Format, Association, Saving (Study/Single), Mode, Other.

  • the words in blue are the current button settings.

Just below the Configuration buttons and their displayed settings, are the 'user-created configuration buttons':

When one of these user-created buttons is clicked, the VistA Imaging Capture window is changed to the saved configuration of that button.

Save Configuration

This dialog can be used to save the current state of the main Capture window as a new configuration or to overwrite an existing configuration.

 

Enter a new or existing configuration name.

 

If an existing configuration name is entered a confirmation dialog will appear.
