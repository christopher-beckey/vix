How to build Filters

When the dialog window is opened [see above]:

  • the name of the Current Filter is displayed in the Title Bar of the window

  • the List area displays the Filters with the Current Filter highlighted [see A above].

 

You can:

  • select a different Filter using the mouse or the Arrow keys

  • apply a Filter by pressing Enter, double clicking the Filter Name or clicking the OK button, the window will then close.

  • press Ctrl+Tab to view the list of Private or Public filters [see B above]

    • modify properties of the current filter:

    • change Image Origin (options are: All Origins, VA or NON-VA)

    • select new date range [see C above]

    • using the Select a Date Range dialog, dates can be customized

Build a Clinical Filter:

  • change the selected packages (Radiology, Medicine, Surgery, etc)

  • select an entry from the drop down lists for: Type, Specialty/SubSpecialty, Procedure/Event [see E above]

Build an Administrative Filter:

   - click the Administrative tab [see D above]

To close the window and apply a Filter

You can:

  • press Enter, double click or click the OK butto.n

  • click the Save button (saves changes to the current Private Filter).

  • click the Save As button (creates a new Private Filter).

 

The Filter is then applied and updates the following:

  • VistA Imaging Display main window.

  • Image Counts are updated ( i.e  “78 of 79 Images match Filter”).

  • Status area displays the Filter Name (Clinical All) and # of images selected.

 

Abstracts window:  

  • Abstracts selected by the Filter are displayed.

  • Filter Name and Filter Count are displayed.

 

Image List window:

  • a label indicating the current filter is displayed.

 

  • a new list is generated.

  • focus is given to the appropriate Tab.

  • filters remain active until changed or the user session is ended.

Testing a Filter before saving it (creating a temporary Custom filter)

You can:

  • open the Filter Add/Edit window (the current filter is displayed)

  • modify the Filter (the Filter name in the window's Title Bar will have “*” appended to it)

OR

  • create a new Filter (the Filter name defaults to ‘Custom’ in the window's Title Bar)

  • select OK  (rather than Save or Save As).

 

The Filter (new or modified) is given the name ‘Custom’ and then applied

 

After reviewing the results of the ‘Custom’ filter the user can save it:

   - open the Image Filter Add/Edit window

   - the current Filter is displayed (in this case, the ‘Custom’ filter)

   - select Save As and enter an appropriate name.
