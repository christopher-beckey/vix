Image List Window - buttons

 

Displays the image for the selected entry.

Allows the user to display the abstract for a selected image/exam on the left side of the Image Listing window.

Displays the report for the selected entry.

Click to display a report for the selected image in a pane at the bottom of the Image Listing window.

Click to increase the vertical length of the window. Click again to reset to the previous size.

Click to resize the columns so that they will display the longest entry.

Click to resize columns so that they all fit in the window.

Click to open the Column Selector dialog, which can be used to control which columns are included in the image list.

Displays the Abstracts window for the patient.

Image Filters can select patient clinical and administrative images/documents - using ranges of dates and image classification (saved by VistA Imaging Capture).

Sets the input focus to the Patient edit box.

  • Click to view the On-Line Help.

  • Right-click to display a shortcut menu or press <Ctrl-Tab>

 

The selected columns are saved along with the 'User Preferences'.
