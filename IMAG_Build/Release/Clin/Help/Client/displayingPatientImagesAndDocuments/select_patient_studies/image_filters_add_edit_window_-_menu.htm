Image Filters Add/Edit window - menu

File menu

 

New - Creates a new Filter

Save - Saves the Filter.

Save As - Saves a Filter under a new name.

Save As Public - Saves a Public Filter, for others to use.

Edit menu

 

Delete Filter - Deletes an existing Filter.

Options menu

 

Clear All - Delete Filters.

Refresh Drop Down Lists - Refresh only Drop Down lists.

Refresh Filter lists - Refresh all Filter lists.

Help menu

 

Image Filter - Accesses the Help topic for Image Filters.
