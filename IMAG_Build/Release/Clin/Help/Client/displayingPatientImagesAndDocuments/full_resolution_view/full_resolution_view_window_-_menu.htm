Vista Imaging Display

Full Resolution View window - menu

 

Shortcut menu  - displays when user right-clicks on the image.

 

Zoom

Resizes the image in the window.

Mouse

Enables changes to the image.

Rotate

Enables rotation of the image:

  • 90 degrees clockwise

  • 90 degrees counter-clockwise

  • 180 degrees

Invert Image

Reverses the colors of the image.

  • Each color is replaced by its color complement.

  • For example, white becomes black, and red becomes blue.

Reset Image

Displays the image -- prior to zoom, rotate and invert operations.

Close Selected Image

Removes the image from the window.

Image Report

Displays the report.

Image Delete,

Image Information,

Image Information Advanced

Are enabled for users with a security key.

Font Size

 

A checkmark is placed next to the current size.

Toolbar

Turns the Toolbars on/off

Show Hints

Displays user Hints when the cursor is over a control.

Goto Main Window
  • Shows Imaging Display main window in front of all open windows.

 

Menus

  • File

Duplicate Selected Image

Create another copy of the image. Each image can be zoomed, inverted and manipulated independently.

Copy Image to Clipboard,

Print Image

 

The user must enter an Electronic Signature to enable these features.

Image Report

Image Information

Image Information Advanced

Image Delete

Close Selected Image,

Close All Images

Removes one or more Images from the window.

 

  • Image

 

Apply to All - all images are changed when options are selected.

Zoom - enables resizing of image within the window.

 

Mouse - enables changes to the image using the mouse.

 

 

Rotate

Enables rotation of the image:

  • 90 degrees clockwise

  • 90 degrees counter-clockwise

  • 180 degrees

Flip Vertical

Display the image with the right side on the left and the left side on the right.

Flip Horizontal

Display the image with the top side on the bottom and the bottom side on the top.

Invert

Reverses the colors of the image.

Reset

Displays the image -- prior to zoom, rotate and invert operations.

Maximize Image

Enable image to fill the entire window.

 

  • View

 

Viewer Settings

Displays the Image Viewer Settings window.

Realign Images

Moves images into consecutive positions, after one or more have been removed from the window.

Lock Image Size
