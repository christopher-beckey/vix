If one of the abstracts is of a Juke Box image, "Display Juke Box Images" must be checked in the View Preferences window to have the abstract displayed. Otherwise, this generic bitmap will be displayed.

 

If the VistA Imaging application cannot connect to the VistA Imaging Network server to retrieve images, the Error graphic will be displayed.

 

If an image from a PACS device has not been copied to the VistA Imaging server, the system will display the PACS graphic.

 

If an image group from a PACS device has not been copied to the VistA Imaging server, the system will display the PACS Group graphic.

 

If an Error occurs for an unknown reason, the Error opening abstract graphic will be displayed.

 

If the Abstract file is corrupt, or cannot be opened by VistA Imaging the Format Error graphic will be displayed.

 

If the Abstract does not exist on the Image Server this graphic will be displayed.
