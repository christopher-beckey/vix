Copy an Image

The copy button is located on the Full Resolution View Window and the Radiology View Window. This will cause the image to be copied to the Clipboard. The Image can then be pasted into a document or drawing using off-the-shelf software such as Word Pad. If the image appears cut-off, resize the image to view all of it.

 

 

The Clinician Agreement for downloaded images Window will open when an Image Print / Copy is attempted. The purpose of the Image copy and an Electronic Signature are required before Print / Copying the image will be enabled.
