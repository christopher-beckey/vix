Image information dialog

An Image Information dialog is available to all users.  The window can be opened using the shortcut menus in the Abstracts, Group Abstracts, Full Resolution View, and Image Listing windows.

 

This dialog can display multiple image information panels The scroll bar on the right side of the window can be used for navigation, and the small "X" buttons in the text areas can be used to close individual frames in the window.
