Image

The Image Report window displays all Image reports and Patient Reports.

 

There is a report button and/or report option in the shortcut menu (right-click on image or list) of all VistA Imaging windows that display images, abstracts, lists of images or list of patient reports.

 

The report contains the long description for the image, if it has been entered, followed by the clinical report.

 

 

VistA Imaging will display the report using the font and font size specified in View Preferences. The font of the report window is saved as a user preference. Select the menu option 'Options|Save settings now' from the Imaging Display window 'Option' menu to save all settings.

 

A list of all opened report windows is appended to the end of the 'Reports' menu on the VistA Imaging Display window. Selecting a report from this menu will redisplay the report.

 

Menu Options for the Image Report Window

 

New Report Window: opens a new Image Report Window.(see Notes)

 

Font: the Windows Font selection dialog window will open. The font, style, point size, effects, color and script are changeable.

 

Print: to print to a network or local printer.

 

Print Setup: opens the standard Windows Print Setup window. This is not recommended for novices. Be sure to change this back to the default printer so the next user will not inadvertently print data to the wrong device.

 

Exit: Closes the Image Report Window.

 

Notes: New Report Window. The next report displayed will be displayed in the new window.  This enables  keeping a report open for viewing later in the session.

 

To open multiple report windows, select 'New Report window'. A new report window will be opened. The next report selected will be displayed in the new report window.

 

Copy a Report:

Click the right mouse button over the text of the report and choose Select All.  Click the right mouse button again and select Copy.  These actions will copy the report to the Windows clipboard.

 

Once the report has been copied to the clipboard it can be pasted into any Windows based word processing program. Optionally, select part of a report displayed in the Report Window with the mouse, click the right mouse button, and select Copy. This will copy the selected part of the report to the Windows clipboard.
