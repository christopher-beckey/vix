Radiology viewer - layout view options

When a study is paged in a Layout View, there are two methods for determining its initial window and level settings.

 

All images can be displayed with the settings selected for the first window.

  • This allows the user to select their own window and level settings
    • either by using the window/level tools
    • or by selecting a CT Preset value.  
  • Settings may be changed when needed.

 

Images may be loaded using the window level settings determined by the selection of "Image Settings" in the Options Menu. This can be either the Device Window Level or the Histogram Window Level.
