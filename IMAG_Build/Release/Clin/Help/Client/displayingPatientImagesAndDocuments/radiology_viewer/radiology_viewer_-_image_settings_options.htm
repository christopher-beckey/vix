Radiology viewer - image settings options

When an image is loaded into the Radiology Viewer, there are two methods for determining its initial window and level settings.

 

In some cases, the radiology device that created the image indicates appropriate window and level settings in the associated image data (also called the image header). By selecting the "Use Device Window Level If Available" option, any settings provided by the device will be used. If none are provided, the histogram option will be used.  This will generally provide a good image display, unless the device does not send proper window and level values.

 

If the Histogram option is selected, then window and level values are selected based on the gray levels detected in the image. This will generally provide a reasonable image display.
