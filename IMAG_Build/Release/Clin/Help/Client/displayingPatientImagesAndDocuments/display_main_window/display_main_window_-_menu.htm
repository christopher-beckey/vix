Select Patient

Opens the Patient Lookup window

Login

Opens the VistA Sign on window

Note: if a connection to VistA exists, a confirmation to disconnect will be necessary before connecting to another VistA System.

Logout

Disconnects the Imaging System from VistA

Remote Login

Opens the VistA Server list. Remote VistA sites can be selected from this list.

Note: The System Manager can modify the Workstation configuration settings.

1 ... 9

Entries: are the most recent patients accessed during this session.

Exit

Disconnects the Imaging System from VistA and closes VistA Imaging.

 

 

Image List Filters

Used to limit the contents of the Image List and Abstracts and Group Abstracts windows.  

 

Image filters, when properly applied,

   => reduce the amount of time it takes to select a specific image for patients with large numbers of images.

View Preferences

Allows user to decide how certain features of the Imaging system will function

Remote Image Views Configuration

Enables VistA Imaging telemedicine features.

Refresh Patient Images

Updates the patient's image list with most recent images.

PreFetch Patient Images

All images for the patient are queued to be copied from the Jukebox to the Magnetic Image server for faster loading of images.  User must have a Security Key.

Save Settings Now

Saves the current settings to VistA.  (click - for more information)

Save Settings on Exit

To save settings when VistA Imaging is closed.

Show Hints

 

Displays Hints when the cursor is over a control in Vista Imaging.

 

Video File Options

Opens the window.

CPRS Link options

Allows user to break the Link to CPRS. It is only enabled if VistA Imaging was started from the CPRS Tools menu.

 

ShortCut Key Legend

Message Log

VistA Imaging maintains a history of messages that have been displayed during the current imaging session. The messages are accessible by any user.

  • They are intended for use by system managers when problems occur.
