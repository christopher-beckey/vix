View Preferences

DETERMINE the NEW explanations

VistA Imaging allows each user to decide how certain features of the Imaging system will function.

 

The View Preferences window has the following options:

  • the 'List Images in Reverse order' dictates the Sort order of the Abstract and Image listing windows.

  • the 'When a patient is selected' section provides options for certain windows to be automatically opened when a patient is selected.

 

When the 'Save' button is clicked the settings from this window, along with the other  window settings that are user definable, will be saved as the User's View Preferences.

 

Changes take effect immediately and remain in effect until VistA Imaging is closed.

Settings for each user are stored in VistA. This feature enables the user to have the same settings on any imaging workstation.
