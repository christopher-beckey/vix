EKG list Options

Select a single Data Type to be displayed in the Patient Tests drop down list box when a patient is selected. Click on the 'radio button' of the test type to list, or click on 'All tests' to list all tests.

 

When the Zoom IN or Zoom OUT buttons are clicked,  the size of the displayed EKG image will be increased or decreased by the Zoom Factor.

 

Use the Zoom Factor slider bar to change the Zoom factor.

 

Note: the Zoom buttons are another way to zoom the image along with the Zoom Slider on the MUSE EKG Display window.

 

Note: The Pages per Screen function and the Date to Start accumulation of tests from are not available.
