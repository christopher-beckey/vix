Patient Tests drop down list box

On the tool bar of the VistA MUSE EKG Display window is the Patient Tests drop down list box which lists the 14 most recent tests on file in the MUSE EKG Database for the current patient.

(graphic is a cutout of the MUSE EKG Display window)

The number of tests listed is displayed to the right of the drop down list box.

 

If not all tests are listed, the count will have a plus sign ( + ) appended to it, to indicate more tests are available.

 

Note: <Click> on the 'More Tests' button to add the next 14 recent tests,or the 'All Tests' button to add all tests to the drop down list box.

 

Note: When focus is on the Patient Tests drop down list box, use the <up> and <down> 'arrow keys' to display the previous or next EKG.

 

Note: The Patient Tests window displays the same Tests list. It is simply an alternate view.
