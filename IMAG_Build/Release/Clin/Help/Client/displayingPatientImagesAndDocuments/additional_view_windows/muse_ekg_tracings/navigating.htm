Navigating

Use the mouse to position the cursor on any control then 'click the left mouse button' to activate the control, or make a selection from a list.

  • Pressing the <tab> key will move the focus to the next control in the window's Tabbing sequence.

  • Pressing <Enter> will activate the control in focus, it's the same as clicking with the left mouse button.

 

If a button is in focus, it will have a dotted rectangle or highlighted border, if an text field or list box has focus, the text will be selected.

 

Example: in the following graphic, the 'Zoom In' button (leftmost button) has focus.

Pressing <tab> will move the focus to the next button to the right. Pressing <shift + tab> will move the focus to the left.

 

When a button on the Toolbar is 'in focus', pressing <right arrow> key is the same as <tab>, and the <left arrow> is the same as <shift + tab>.

 

When a control is in focus, pressing <Enter> or clicking the left mouse button give the same result.

 

'Browsing' the Patient's EKG tests can be accomplished in two ways:

(1) Move focus to the Previous or Next Test buttons by tabbing or clicking with the mouse, then repeatedly press <Enter>.

(2) Move focus to the Patient Tests drop down list box, and press the <down arrow> or <up arrow>.
