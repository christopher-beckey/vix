Video Player

Motion video files can be saved as part of the patient record using VistA Imaging.

 

(Only video files of types .AVI and .MPG are currently supported.)

 

When the selected image is a motion video clip, the Video Player window is opened.

Use the tool bar buttons to play, pause, stop and rewind.

 

Use the slider bar to browse the video clip, frame by frame.

 

'Start' and 'End' points for a video loop can be set for continuous play of all or part of the file.

 

(see also: Button, Menu and ShortCut menu descriptions of the Video Player window)

 

Button Descriptions of the Video Player Window

(left to right above)

Play, Pause, Stop, Rewind, Progress bar (slider bar).

 

The graphic above depicts how the window controls will be seen when the menu Option 'Progress Bar' is checked, a Start Point and an End Point have been selected, and "Increment in Frames" is checked.

 

When the slider bar is active, the video might seem a little 'choppy', 'hesitant'. this is because extra processing is required to synchronize the slider control with the current frame of the video clip. When 'Progress bar' is unchecked, this 'hesitation' will be less noticeable.

 

(left to right above)

Play, Pause, Stop, Fast Forward, Rewind, one Frame forward, one Frame backward.

The graphic above depicts how the window controls will be seen when the menu option 'Progress Bar' is not checked.

 

 

Menu Descriptions of the Video Player Window

 

'Image Report' displays the associated VistA Report for this image.

'Exit' will close the Video Player window.

 

'Zoom Out' will decrease the size of the window.

'Zoom In' will increase the size of the window.

'Default' will reset the size of the display to the default ( original )

  • Note: the default size is the best size for picture quality.

'Set Start Point' and "Set End Point" at different places in the video, will select a region of the video clip. When a region is selected, only that part of the video clip will be shown, when 'play' is started.

'Clear Start/End' will clear the selected region.

 

'Play' will start the video in motion from the current position of the slider bar.

'Stop' will stop the video.

'Rewind' to rewind the video clip to the beginning.

 

Shortcut menu of the Video Player window

Press <Ctrl - Tab> or right click using the mouse to display the shortcut menu.

'Image Report' displays the associated VistA Report for this image.

'Set Start Point' and "Set End Point" at different places in the video, will select a region of the video clip. When a region is selected, only that part of the video clip will be shown, when 'play' is started.

'Clear Start/End' will clear the selected region.

Select 'Video Loop' for continuous play of the whole video clip, or the selected region.

'Goto Main Window' changes application focus to the Main Imaging Display window

 

All options of the shortcut menu are also on the main menu for the Video Player window.
