vista imaging display

Customizing layout

Up to 64 images can be loaded at once (depending on the layout requested by the user)

  • if there are more images loaded than the current layout can accommodate, some images will be hidden but are still available for display. Scrollbars located on the bottom and right edges of the window can be used to display the additional images.

  • depending on the user layout, the image may occupy part or all of the display area

 

Layout is expressed in columns and rows

 

1x2 = 1 column and 2 rows

2x4 = 2 columns and 4 rows

  • the first displayed image is shown in the upper left corner of the display window

  • additional images opened are arranged from top to bottom, then from left to right.

  • if the number of loaded images exceeds the current layout, you can use the scrollbars to display the remaining images.

 

Lock Image Size feature

If the window is resized when multiple images are displayed (for instance, 2x4 layout) -- the individual images will automatically contract or expand to fill the resized window.

 

If View => Lock Image Size is checked the Full Resolution View window can be resized but the individual images displayed will remain the same size.
