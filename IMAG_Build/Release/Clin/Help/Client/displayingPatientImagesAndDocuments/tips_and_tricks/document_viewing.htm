Document viewing

Documents

When a "multi-page" document is displayed, the image is shown in a virtual stack, with only the first page visible. The pages can be displayed using the multi-page controls on the toolbar.

 

The muti-page controls are enabled only when a multi-page image is selected.

 

Documents - 'Quick View'

(1) The document with the blue selection box is a "bit" small to read.

  • click the  Zoom image to actual size button (1 pixel to 1 pixel)

  • the multi-page document is selected and fills the entire window.

 

 

(2) Use the Zoom control for additional magnification.
