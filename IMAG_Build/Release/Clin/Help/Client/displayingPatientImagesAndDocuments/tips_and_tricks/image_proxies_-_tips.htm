Image proxies - tips

See also: Image Proxies

 

When the total number of displayed images exceeds the value of the Max # Images in the Viewer Settings dialog (the menu of the main Display window)

  • A white rectangle is shown, in place of the image.

  • The image description is shown inside the white rectangle.

  • To display the image that the proxy represents -- click the proxy.

 

Why use proxies

Advantages:

  • proxies will shorten the time for images to load on the screen

  • users will see the image description before loading the image

  • conserve available computer memory

 

When to use proxies:

  • on a slow network.

  • using a workstation with limited memory.

 

How to use proxies:

  • enter a low value in the Viewer Settings dialog.

 

To avoid using proxies

Enter 64 (the highest value accepted) in the Max # Images box in the Viewer Settings dialog.
