General features

Also see => Tips & Tricks for: document viewing, customizing layout, image proxies

 

The Full Resolution View window displays:

  • medical images and scanned documents simultaneously

  • non-DICOM images are displayed in this window

  • scanned documents can be single and multi-page

  • the images / documents can be from different dates

  • the total number of loaded images is shown on the left side just above the display area

  • the label, at the top of each image, identifies which procedure it is associated with.

 

Switch between single & all images

A quick way to switch between displaying a single image and all loaded images, using the toolbar,

o click the (Display a single image) and

  • then click the (Display all images) buttons

 

Remove & Refresh images

When mutiple images / documents are displayed in the window:

  • the user may remove single images or groups

  • removal of images will leave vacant areas

 

Hold down the CTRL key and select 2 images.

  • Click the Close Selected Images button.

 

(1) Both images are removed from the Full Resolution View window

 

  • when the user refreshes the window (for instance, by sizing the window) the images will be rearranged automatically to fill vacant areas. Another method of refreshing the window is to select View => Realign Images

.

(2) The remaining images are rearranged to fill the window

Inverting an image

Select an image, then click to return the image to its original state click the icon again.

 

When viewing a black\white image,

  • toggling the image back and forth aids the user to view subtle feature.

When viewing a color image with cellular features,

  • toggling the image back and forth aids in seeing the cellular boundaries.

 

Zoom in / out

Select an image then use the Zoom control on the toolbar,

  • for quick changes, use the mouse to drag the slider left or right

  • to change by 10% increments, click to the left or right of the slider bar.

  • to change by 1% increments, click the control and use the arrow keys (right or left).

  • to change by 20% increments, right-click the image, click Zoom and then choose Zoom In or Out.
