Abstracts and Group Abstracts-shortcut menu

See also: Abstracts and Group Abstracts

 

Shortcut menu is displayed by right-clicking on an abstract.

 

 

Open Image

View the full resolution image.

Image Report

View the Image Report window.

Image Delete...

Delete the image.

Image Information...

View information related to the image.

Image Information Advanced...

View additional details related to the image (enabled for users with a security key).

Cache Images

Enable images to be rapidly displayed.

Image List Filters...

Opens the Image Filters Add/Edit dialog.

Refresh

Reopen image and will undo changes made by the user.

Resize the Abstracts...

Allows the user to drag the edges of an abstract to change the size of all displayed abstracts.

Font Size

Change the size of the titles under the abstracts.

Toolbar

Reveals and hides the tool bar for the Abstracts window.

Show Hints

Provide additional information about images.

Go to Main Window

Moves focus to the VistA Imaging Display  Window.

Help...

Displays the On-Line Help for the Abstract window.

  • The F1 key will also display the Help window.
