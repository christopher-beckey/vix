VistA Imaging TeleReader

Selection of Studies

The main TeleReader window contains the remote connections status bar, the read and unread list of studies, details about studies, and a status bar.

 

A study highlighted in:

  • Red indicates the Urgency of that item is STAT and should be resulted quickly.  

  • Blue indicates that item has been locked by the current user.  

  • Black are unlocked and can be resulted by the current user.  

  • Gray studies are items which the current user cannot result (either the item is locked or the current user is not logged into the reading site for that item).

 

When the user selects a study in the Read or Unread list, the details of that study are displayed at the bottom of the TeleReader.  Studies which have the status of “Unread” and can be read by the user can be locked by clicking on the Lock button.  The Lock button will be disabled when the user selects a study which cannot be read by the user.  Clicking the Lock button causes the status of the study to change to “Locked” and the patient context for desktop applications to change to the selected patient.  The study can be resulted by viewing associated images in Clinical Display and entering the result in CPRS.  When resulted, studies move from the Unread List to the Read list.

 

The user can select any study in the read or unread list and click the View button to view the images and information in the study using Clinical Display and CPRS.
